### 속성 선택자의 종류
1.선택자1[속성명]
```
<head>
    <style>
        .attr1[test]{background-color: skyblue;}
    </style>
</head>
<body>
    <h2 class="attr1" test="title">1.선택자1[속성명]</h2>
    <div class="attr1" test="content">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명을 지니고 있는 모든 태그를 검색

2.선택자1[속성명=속성값]
```
<head>
    <style>
        .attr2[test=title]{background-color: yellowgreen;}
    </style>
</head>
<body>
    <h2 class="attr2" test="title">2.선택자1[속성명=속성값]</h2>
    <div class="attr2" test="content">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명이 속성값과 일치하는 태그들 검색

3.선택자1[속성명~=속성값]
```
<head>
    <style>
        .attr3[test~=content]{background-color: beige;}
    </style>
</head>
<body>
    <h2 class="attr3" test="title attr3">3.선택자1[속성명~=속성값]</h2>
    <div class="attr3" test="attr3 content">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명에 대한 속성값들이 공백을 구분자로 해당 속성값을 가지고 있는 태그 검색

4.선택자1[속성명|=속성값]
```
<head>
    <style>
        .attr4[test|=attr4]{background-color: orange;}
    </style>
</head>
<body>
    <h2 class="attr4" test="attr4-title">4.선택자1[속성명|=속성값]</h2>
    <div class="attr4" test="attr4-content">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명에 대한 속성값이 '속성값'이거나 '속성값-' 시작하는 태그들 검색

5.선택자1[속성명^=속성값]
```
<head>
    <style>
        .attr5[test^=t]{background-color: aliceblue;}
    </style>
</head>
<body>
    <h2 class="attr5" test="title">5.선택자1[속성명^=속성값]</h2>
    <div class="attr5" test="content">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명에 대한 값이 '속성값'만 존재하거나 '속성값'으로 시작하는 태그들 검색

6.선택자1[속성명$=속성값]
```
<head>
    <style>
        .attr6[test$=6]{background-color: pink;} //속성값 위치에 숫자만 들어올수 없으므로 적용X<br>
        .attr6[test$=attr6]{background-color: pink;} //적용O
    </style>
</head>
<body>
    <h2 class="attr6" test="title_attr6">6.선택자1[속성명$=속성값]</h2>
    <div class="attr6" test="content_attr6">
</body>
```
- 선택자1과 일치하는 태그들 중 속성명에 대한 값이 '속성값'이거나 '속성값'으로 끝나는 태그들 검색\

### 가상 클래스 선택자
가상 클래스 선택자란?
- 가상의 상황을 예측하고 미리 디자인하고자할 때 사용하는 선택자입니다.

1.:hover 선택자
```
<head>
    <style>
        .title:hover{background-color: beige;}
    </style>
</head>
<body>
    <h3 class="title">1. :hover 선택자</h3>
</body>
```
- 문법) 선택자1:hover{속성들...}
- 선택자1과 일치하는 태그들 영역안에 마우스가 오버된 경우 디자인을 적용합니다.

2.:active 선택자
```
<head>
    <style>
        .title:active{color: red;}<br>
    </style>
</head>
<body>
    <h3 class="title">2. :active 선택자</h3>
</body>
```
- 문법) 선택자1:active{속성들...}
- 선택자1과 일치하는 태그를 마우스로 누르고 있는 경우 감지하는 선택자입니다.

3.:focus 선택자
```
<head>
    <style>
        .text:focus{background-color: khaki;}
    </style>
</head>
<body>
    <input type="text" class="text">
    <input type="text" class="text">
</body>
```
- 문법) 선택자1:focus{속성들...}
- 선택자1번과 일치하는 태그들 중 현재 키보드 입력 독점권(포커스)를 얻은 태그 검색

4.:checked 선택자
```
<head>
    <style>
        input[type=checkbox]:checked{background-color: khaki;} // 브라우저에서 제공하는 checkbox, radio는 디자인 변경X
        input[type=checkbox]:checked+span{background-color: khaki;} // 변경O
    </style>
</head>
<body>
    <input type="checkbox"><span>체크</span><br>
</body>
```
- 문법) 선택자1:focus{속성들...}
- 선택자1번과 일치하는 태그들 중 현재 키보드 입력 독점권(포커스)를 얻은 태그 검색

5.:link 선택자
```
<head>
    <style>
        a:link{color: orange;}
    </style>
</head>
<body>
    <a href="https://www.naver.com">네이버로 이동</a>
</body>
```
- 문법) 선택자1:link{속성들...}
- 선택자1과 일치하는 태그들 중 브라우저로 방문한적 없는 경로에 대한 링크들 검색합니다.

6.:visited 선택자
```
<head>
    <style>
        a:visited{color: skyblue;}
    </style>
</head>
<body>
    <a href="https://www.naver.com">네이버로 이동</a>
    <a href="https://www.google.com">구글로 이동</a>
</body>
```
- 문법) 선택자1:visited{속성들...}
- 선택자1과 일치하는 태그들 중 브라우저로 방문한적 있는 경로에 대한 링크들 검색합니다.

### 가상 요소 선택자
가상 요소 선택자란?
- 태그로 감싸지 않은 가상의 영역을 찾아 디자인 할 때 사용합니다.

1.::first-letter 선택자
```
<head>
    <style>
        h1::first-letter{color: red;}
    </style>
</head>
<body>
    <h1>가상 요소 선택자</h1>
</body>
```
- 문법) 선택자1::first-letter{속성들...}
- 선택자1과 일치하는 태그들의 영역 안 첫 글자를 찾아 디자인합니다.

2.::first-line 선택자
```
<head>
    <style>
        .line::first-line{color: violet}
    </style>
</head>
<body>
    <div class="line">
        문법) 선택자1::first-link{속성들....} <br>
        - 선택자1과 일치하는 태그들의 영역 안 첫 줄을 찾아 디자인합니다.<br>
    </div>
</body>
```
- 문법) 선택자1::first-link{속성들....} <br>
- 선택자1과 일치하는 태그들의 영역 안 첫 줄을 찾아 디자인합니다.

3.:first-child 선택자
```
<head>
    <style>
        p:first-child{background-color: yellow;}
    </style>
</head>
<body>
    <p>문법) 선택자1:first-child{속성들...}</p>
    <div>
        - 선택자1과 일치하는 태그들 중 첫번째 자식에 해당하는 태그들 검색합니다.
    </div>
</body>
```
- 문법) 선택자1:first-child{속성들...}
- 선택자1과 일치하는 태그들 중 첫번째 자식에 해당하는 태그들 검색합니다.

4.:last-child 선택자
```
<head>
    <style>
        p:last-child{background-color: beige;} //적용X <br>
        div:last-child{background-color: beige;} //적용o
    </style>
</head>
<body>
    <p>
        문법) 선택자1:last-child{속성들...}
    </p>
    <div>
        선택자1과 일치하는 태그들 중 부모 기준 마지막 자식 태그에 해당하는 태그 검색합니다.
    </div>
</body>
```
- 문법) 선택자1:last-child{속성들...}
- 선택자1과 일치하는 태그들 중 부모 기준 마지막 자식 태그에 해당하는 태그 검색합니다.

5.:nth-child(자연수, 수열) 선택자
```
<head>
    <style>
        div:nth-child(2){background-color: beige;}
    </style>
</head>
<body>
    <div>
        문법) 선택자1:nth-child(자연수, 수열){속성들...}
    </div>
    <div>
        - 선택자1과 일치하는 태그들 중 괄호안 자연수 또는 수열번째에 해당하는 자식인 태그를 검색합니다.
    </div>
    <div>
        예) .ele05 div:nth-child(2){background-color: beige;}
    </div>
</body>
```
- 문법) 선택자1:nth-child(자연수, 수열){속성들...}
- 선택자1과 일치하는 태그들 중 괄호안 자연수 또는 수열번째에 해당하는 자식인 태그를 검색합니다.

6.::before 선택자
```
<head>
    <style>
        div::before{content:'-'; color: orangered;}
    </style>
</head>
<body>
    <div>
        문법) 선택자1::before{content:''; 속성들...}
    </div>
    <div>
        선택자1과 일치하는 태그들의 가상의 첫번째 자식 영역을 만들어 줍니다.<br>
        content 속성 값이 가상의 자식 영역 안 내용이 됩니다.
    </div>
</body>
```
- 문법) 선택자1::before{content:''; 속성들...}
- 선택자1과 일치하는 태그들의 가상의 첫번째 자식 영역을 만들어 줍니다.
- content 속성 값이 가상의 자식 영역 안 내용이 됩니다.    

7.::after 선택자
```
<head>
    <style>
        div:nth-child(3)::after{content: '★'; color: yellow;}
    </style>
</head>
<body>
    <div>
        문법) 선택자1::after{content:''; 속성들...}
    </div>
    <div>
        - 선택자1과 일치하는 태그들의 마지막 자식 요소 영역을 만들어 줍니다.<br>
        - content 속성 값을 생성된 마지막 자식요소의 내용을 삽입해줍니다.
    </div>
    <div>
        예) .ele07 div:nth-child(3)::after{content: '★'; color: yellow;}
    </div>
</body>
```
- 문법) 선택자1::after{content:''; 속성들...}
- 선택자1과 일치하는 태그들의 마지막 자식 요소 영역을 만들어 줍니다.
- content 속성 값을 생성된 마지막 자식요소의 내용을 삽입해줍니다.

### 텍스트 관련 속성
1.text-indent
```
<head>
    <style>
        .ex01 .p01{text-indent:16px;}
        .ex01 .p02{text-indent:1em;}
        .ex01 .p03{text-indent:1rem;}
    </style>
</head>
<body>
    <div class="ex01" style="font-size: 20px;">
    <p class="p01">px 단위로 들여쓰기 하기(.ex01 .p01{text-indent:16px;})</p>
    <p class="p02">em 단위로 들여쓰기 하기(.ex01 .p02{text-indent:1em;})</p>
    <p class="p03">rem 단위로 들여쓰기 하기(.ex01 .p03{text-indent:1rem;})</p>
</body>
```
- 문단의 첫글자 들여쓰기를 제어할 수 있는 css 속성입니다.

사용할 수 있는 속성 값의 단위
- 1.px : 고정길이로 정해진 px 길이 만큼 들여쓰기를 합니다.
- 2.em : 부모의 글자크기를 기준으로 배수 단위 입니다.(부모가 20px 글자크기이면 1e->20px)
- 3.rem : 최상위 글자크기를 기준으로 배수 단위입니다.(부모가 20px 글자크기여도 html에 따로 글자크기가 없다면 1rem=16px)

2.text-align
```
<head>
    <style>
        .p01{text-align:left;}
        .p02{text-align:right;}
        .p03{text-align:center;}
        .p04{text-align:justify;}
    </style>
</head>
<body>
    <p class="p01">왼쪽 정렬(.ex02 .p01{text-align:left;})</p>
    <p class="p02" style="width: 400px;">오른쪽 정렬(.ex02 .p02{text-align:right;})</p>
    <p class="p03">가운데 정렬(.ex02 .p03{text-align:center;})</p>
    <p class="p04" style="width: 500px; border: 1px solid black;">
</body>
```
- 영역안 텍스트들의 가로 정렬을 제어할 수 있습니다.
- 인라인 성질 태그들은 부모의 text-align 속성의 영향을 받아 배치됩니다.
- 위 strong, input은 부모 너비 기준 배치
- 블럭 성질 태그들은 부모의 text-align 속성 값이 상속되어 자식의 너비 기준 배치가 됩니다.
- 위 div는 자신의 너비 기준 부모의 text-align 속성값 배치

속성 값 종류
1. left : 기본 값으로, 텍스트를 왼쪽 정렬합니다.
2. right : 태그의 너비 기준 오른쪽 정렬합니다.
3. center : 태그의 너비 기준 중앙 정렬합니다.(너비를 지정하지 않은 경우 브라우저 정중앙 배치)
4. justify : 영역 안 텍스트들의 왼쪽과 오른쪽 열이 균일하게 맞아 떨어지도록 텍스트를 정렬합니다.

3.text-decoration
```
<head>
    <style>
        .p01 a{text-decoration: none;}
        .p02{text-decoration: underline;}
        .p03{text-decoration: overline;}
        .p04{text-decoration: line-through;}
    </style>
</head>
<body>
    <p class="p01">
        <a href="#">none 예제 p태그(.ex03 .p01{text-decoration: none;})</a>
    </p>
    <p class="p02">underline 예제 p태그(.ex03 .p02{text-decoration: underline;})</p>
    <p class="p03">overline 예제 p태그(.ex03 .p03{text-decoration: overline;})</p>
    <p class="p04">line-through 예제 p태그(.ex03 .p04{text-decoration: line-through;})</p>
</body>
```
- 영역 안 텍스트들의 선 위치를 지정하는 속성입니다.

속성값의 종류
1. none : 선 제거
2. underline : 밑줄
3. overline : 윗줄
4. line-through : 텍스트 중앙선

### 글꼴 관련 속성
1.font-family(글꼴)
```
<head>
    <style>
        .p01{font-family: a, b, serif;}
        .p02{font-family: a, b, sans-serif;}
        .p03{font-family: a, b, monospace;}
        .p04{font-family: a, b, cursive;}
    </style>
</head>
<body>
    <p class="p01">serif 형태의 폰트</p>
    <p class="p02">sans-serif 형태의 폰트</p>
    <p class="p03">monospace 형태의 폰트</p>
    <p class="p04">cursive 형태의 폰트</p>
</body>
```
- 특정 영역 안 글꼴을 제어할 때 사용하는 속성입니다.
- 폰트 변경시 대체 폰트 3가지 정도 함께 작성할 것을 권장합니다.
- 콤마(,) 구분자로 폰트명들을 속성값으로 나열합니다.
- 마지막 세번째 폰트는 브라우저에서 제공하는 서체별 폰트명인 generic 폰트를 작성할 것을 권장합니다.
- 폰트 명이 한글인 경우 또는 폰트명에 공백이 포함된 경우에는 속성값을 ''감싸야 합니다.

generic 폰트 종류
1. serif : 글자 끝에 엣지가 있는 바탕체와 유사한 서체
2. sans-serif : 글자 끝에 엣지가 없는 고딕체와 유사한 서체
3. monospace : 글자간 크기가 균일한 형태의 서체
4. cursive : 필기체 계열, 궁서체와 유사한 서체

2.font-size(크기)
```
<head>
    <style>
        .sizeEx p:nth-child(1):active{font-size: 16px;}
        .sizeEx p:nth-child(2):active{font-size: 2em;}
        .sizeEx p:nth-child(3):active{font-size: 2rem;}
    </style>
</head>
<body>
    <div class="sizeEx" style="font-size: 20px;">
        <p>마우스 클릭 시 글자 크기를 16px로 작게 적용합니다.</p>
        <p>마우스 클릭 시 글자 크기를 부모 기준 2배 적용합니다.</p>
        <p>마우스 클릭 시 글자 크기를 최상위 기준 2배 적용합니다.</p>
    </div>
</body>
```
- 특정 영역 안 글자 크기를 제어하는 속성입니다.

자주 사용하는 단위) 
1. px : 픽셀 단위 글자 크기 제어 (고정크기)
2. em : 부모 기준 배수 단위 (부모 20px -> 1em -> 20px)
3. rem : 최상위 기준 배수 단위 (부모 20px, html 16px -> 1rem -> 16px)

3.font-weight(굵기)
```
<head>
    <style>
        p:nth-child(1){font-weight: 100;}
        p:nth-child(2){font-weight: 400;}
        p:nth-child(3){font-weight: 700;}
        p:nth-child(4){font-weight: 900;}
    </style>
</head>
<body>
    <p>100 굵기로 적용해보세요.</p>
    <p>400 굵기로 적용해보세요.</p>
    <p>700 굵기로 적용해보세요.</p>
    <p>900 굵기로 적용해보세요.</p>
</body>
```
- 특정 영역 안 글자 굵기를 제어하는 속성입니다.
- 속성값은 100-900까지 단위 없이 100씩 증가 범위로 사용합니다.(100,200,...,900)
- h태그, b태그, strong 태그는 기본 700 굵기를 갖는다.
- 기본 굵기는 400 굵기이다.
- 만약 100 단위가 아닌 10의 단위를 작성한다면 반올림을 적용합니다. 예) 430 -> 400, 450 -> 500

정해진 키워드 속성값 종류
1. bold : 700 굵기에 대한 속성값<br>
2. normal : 400 굵기에 대한 속성값<br>
3. lighter : 부모 보다 얇게(300 정도 얇게)<br>
4. bolder : 부모 보다 굵게(200 정도 굵게)<br>
