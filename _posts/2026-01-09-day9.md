#### 문자 데이터 타입
CHAR(n) : 1 ~ 255 바이트 | 고정길이 문자형 <br>
VARCHAR(N) : 1 ~ 65535 바이트 | 가변길이 문자형 <br>
TEXT : 1 ~ 65535 바이트 | N 크기의 TEXT 데이터 값 <br>
ENUM : 1 또는 2 | 최대 65535개의 열거형 데이터 값 <br>
SET : 1, 2, 3, 4, 8 | 최대 64개의 서로 다른 데이터 값

#### 비교 연산자
- = : 같다 <br>
- <> : 다르다 <br>
- < : 작다 <br>
- \> : 크다 <br>
- <= : 작거나 같다 <br>
- \>= : 크거나 같다 <br>
- IS NULL : NULL과 같다 <br>
- IS NOT NULL : NULL이 아니다 

#### 논리 연산자
- AND : 모든 조건을 만족해야 검색
- OR : 여러 조건 중 한 가지만 만족해도 검색
- NOT : 조건을 만족하지 않는 것만 검색

#### LIKE 키워드
- LIKE '데이터%' : 데이터로 시작하는 문자열 (데이터로 시작하기만 하면 길이는 상관없음)
- LIKE '%데이터' : 데이터로 끝나는 문자열 (데이터로 끝나기만 하면 길이는 상관없음)
- LIKE '%데이터%' : 데이터가 포함된 문자열
- LIKE '데이터___' : 데이터로 시작하는 6자길이의 문자열
- LIKE '__데이터%' : 세 번째 글자의 시작이 '데이터'인 문자열 (길이 상관없음)

#### 집계함수
- SUM : SUM() - 합계를 구한다.
- AVG : AVG() - 평균을 구한다.
- MIN : MIN() - 최솟값을 구한다.
- MAX : MAX() - 최댓값을 구한다. 
- COUNT : COUNT() - 행의 갯수를 센다.
- COUNT DISTINCT : COUNT(DISTINCT) - 행의 갯수를 센다. (중복은 1개만 인정) 

\*집계 할때 일반적인 항목과는 같이 사용x

#### 윈도우 함수
- MySQL 8 버전 이상 부터 지원
- 행과 행간을 비교, 연산, 정의하기 위한 함수
```
SELECT
   WINDOW_FUNCTION(ARGUMENTS) OVER (PARTITION BY 컬럼 ORDER BY 컬럼 오름차순|내림차순)
FROM 
   테이블명;
```
\- ARUMENTS : 윈도우 함수에 따라 0 ~ N개의 인수를 설정 <br>
\- PATITION BY : 전체 집합을 기준에 의해 소그룹 나눔 <br>
\- ORDER BY : 컬럼 정렬 <br>
\- rank() over : 랭크를 정렬하여 보여줌

####  GROUP BY, HAVNIG
데이터를 그룹화하여 그룹별로 조회

```
SELECT
    p.vend_cd ,
    count(p.prod_cd) AS '상품등록수'
FROM 
    products p
GROUP BY p.vend_cd   //데이터를 그룹화
HAVING count(p.prod_cd) > 20;    //그룹을 조건에 맞게 필터링

```

Having 조건 - 조건에 맞게 그룹화
