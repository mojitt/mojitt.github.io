#### 2. id(객체)
- 객체의 주소를 반환하는 내장함수
- 파이썬에서는 메모리를 효츌적으로 관리하기 위하여 간단한 정수와 문자열은 같은 주소를 가집니다.

```
num1 = 10
num2 = 10
str1 = 'a'
list1 = [1,2,3]

print(id(num1)) # 140731194627272(메모리 주소값)
print(id(num2)) # 140731194627272
print(id(10)) # 140731194627272, 리터럴 값이 같으면 주소값도 같음
print(id(str1)) # 140731194691552
print(id('a')) # 140731194691552, 리터럴 값이 같으면 주소값도 같음
print(id(list1)) # 2554443006720
print(id([1,2,3])) # 2554442849024, 리스트는 리터럴 값이 같아도 주소값은 다름

def func1(arg):
    print(arg)
    print(id(arg))
    
print('-'*40)

list1 = [4,5,6]
print(id(list1))
func1(list1)
```

## 내장 라이브러리
#### 1. 날짜 객체
- 최상위 패키지 명은 datetime

```
import datetime
```

- date객체를 사용하면 년-월-일 객체를 생성할 수 있습니다.

```
print(datetime.date(2026,3,13))
print(datetime.date.today())
```

- datetime객체를 사용하면 년-월-일-시-분-초 객체를 생성할 수 있습니다.
- 만약 31보다 넘치는 날짜 또는 24시 이상 입력시 value error 발생

```
print(datetime.datetime(2026,3,13))
print(datetime.datetime.now()) # today()도 가능

today = datetime.datetime.now()
print(today.year)
print(today.month)
print(today.day)
print(today.weekday()) # 요일, 0부터 월요일 (0 ~ 6)
print(today.isoweekday()) # 요일, 1부터 월요일 (1 ~ 7)

today = today.replace(month=2) # 해당 날짜를 수정한 새로운 날짜 객체 반환
print(today)

print(today.strftime('%Y-%m-%d'))

'''
%a : 영문 요일 단축 표현 (Mon)
%A : 영문 요일 전체 (Monday)
%b : 영문 월 단축 표현 (Jan)
%B : 영문 월 전체 (January)
%c : 영문 요일, 월, 일 시:분:초, 연도
%d : 일(날짜)
%H : 24시 기준 시간
%I : 12시 기준 시간
%j : 1년 기준 누적 날짜
%m : 숫자 월
%M : 분
%p : AM / PM
%s : 초
%U : 1년중 누적 주수
$w : 숫자 요일(0: 월요일 6: 토요일)
%W : 1년중 누적 월요일
%Y : 년도 4자리
%X : 타임존
%y : 년도 2자리
'''
```

- 서로 다른 두 날짜의 연산을 구할 때에는 두 날자로 더하기 빼기 연산하면 됩니다.
- 연산의 결과는 timedelta 타입으로 반환 합니다.
- timedelta 객체는 days, seconds, microseconds 필드를 가지고 각 의미에 맞는 정보를 반환합니다.

```
today = datetime.datetime.now()
d_day = datetime.datetime(2026,6,23)

diff = d_day - today
print(diff.days)

# 특정날짜의 며칠 전후를 알고 싶다면 timedelta 객체를 일 수 만큼 대입하여 생성하고
# 날짜객체에 더하기 빼기 연산 하면 됩니다.
day = datetime.timedelta(days=10)

new_day = today + day
print(new_day)

new_day = today - day
print(new_day)
```

##### 3_1) zip_longest(반복객체1, 반복객체2, fillvalue=기본값)
- 길이가 긴 반복객체를 기준으로 인덱스 짝 튜플 목록을 반환합니다. (fillvalue= 길이가 모자르는 원소의 기본값)

```
list1 = ['a','b','c','d']
list2 = [1,2,3]

print(list(zip(list1, list2)))
print(list(itertools.zip_longest(list1, list2, fillvalue=0)))
```

##### 3_2) permutations(반복객체, 원소 조합 갯수)
- 반복객체가 원소 조합 갯수를 가질 수 있는 모든 경우의 수를 튜플로 목록 반환합니다.

```
'''
[1,2,3] -> 두자리 경우의 수 -> (1,2),(1,3),(2,1),(2,3),(3,1),(3,2)
'''

list1 = [1,2,3]
print(list(itertools.permutations(list1, 2)))
```

##### 3_3) combination(반복객체, 원소 조합 갯수)

```
'''
[1,2,3] -> 두자리 경우의 수 -> (1,2),(1,3),(2,3)
'''

list1 = [1,2,3]
print(list(itertools.combinations(list1, 2)))
```

#### 6. json
- json 파일을 불러와 딕셔너리 구조로 변화할 때 사용하는 라이브러리

```

# json 파일을 읽어와 딕셔너리로 형변환 할 때에는 json.load(파일객체)
with open(r'C:\dev\workspace\python\workspace\03.13\data.json') as f:
    json_data = json.load(f)
    print(type(json_data))
    
    
dict_data = {'name':'홍길동', 'age':30, 'address':'덕진구 기린대로 499'}

# 딕셔너리를 json 파일로 만들 때 json.dump(딕셔너리객체, 파일)
with open(r'C:\dev\workspace\python\workspace\03.13\data2.json', 'w') as f:
    json.dump(dict_data,f)
    
# json 문자열을 딕셔너리 객체로 만들 때 json.loads(문자열)
dict_str = '{"name":"홍길동", "age":30, "address":"덕진구 기린대로 499"}'

dict1 = json.loads(dict_str)
print(dict1)
print(type(dict1))

# 딕셔너리를 json형식 문자열로 만들 때 json.dumps(딕셔너리)
dict1 = {'name':'홍길동', 'age':30, 'address':'덕진구 기린대로 499'}

print(json.dumps(dict1, ensure_ascii=False))
```
