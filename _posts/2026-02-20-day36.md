#### bool 타입
- True, False를 갖는 자료형
- 주로 연산의 결과값으로 반환되는 자료형입니다.
- 리터럴 표기시 True, False 로 표기합니다.

```
a = True
#b = true # 파이썬에서는 true 리터럴은 True 입니다.

b = False
#b = false # 파이썬에서는 false 리터럴은 False 입니다.

result = a == b # 주로 비교, 논리 연산의 결과 반환 타입입니다.
print(result) # False


# 파이썬에서 참을 의미하는 것들 (truthy)

a = 1           # 숫자는 0이 아니면 true를 의미합니다.
a = [1,2,3,4]   # 리스트는 원소가 있다면 true를 의미합니다.
a = (4,)        # 튜플은 원소가 있다면 true를 의미합니다.
a = {1:'a'}     # 딕셔너리는 원소가 있다면 true를 의미합니다.
a = {1}         # 셋(set)은 원소가 있다면 true를 의미합니다.
a = 'a'         # 문자열은 빈문자열이 아니면 true를 의미합니다.

if a : print('참') # a 변수가 truthy인 경우 출력


# 파이썬에서 참을 의미하는 것들 (falsy)

a = False
a = 0       # 숫자가 0인 경우 false를 의미합니다.
a = ''      # 문자열이 빈문자열인 경우 false를 의미합니다.
a = []      # 리스트의 원소가 없는 경우 false를 의미합니다.
a = ()      # 튜플의 원소가 없는 경우 false를 의미합니다.
a = {}     # 딕셔너리 또는 set의 원소가 없는 경우 false를 의미합니다.

if not a : print('거짓') # a 변수가 truthy인 경우 출력

# bool(자료형) 함수를 사용하여 자료형이 true인지 false인지 알 수 있습니다.
print(bool('a')) # True
```


#### 파이썬에서 변수 사용하기
- 데이터들의 주소를 참조하는 역할
- 선언 시 데이터 타입을 필요로 하지 않습니다.
- 변수 선언 : 변수명 = 값
- 변수는 데이터의 주소를 참조하는 형태로 파이썬에서 모든 데이터는 객체입니다.

#### 변수명 선언 시 규칙
1. 대소문자 구분
2. 영문으로 작성
3. 숫자로 시작 할 수 없음
4. '_' 사용가능
5. 예약어는 사용x
6. 변수명 사이 공백 사용x
7. '_'를 제외한 특수문자 사용x

```
number7 = 7

#7number = 8 # 변수명이 숫자로 시작 할 수 없습니다.

a = 12345678910 # 같은 숫자다 하더라도 객체이므로 다른 주소를 나타냅니다.
b = 12345678910 # 숫자와 문자열 타입은 작은 범위의 숫자 또는 간단한 문자열은 값이 같으면 같은 주소를 참조합니다. 숫자(-5~256)까지는 같은 주소를 가진다.
print(id(a) is id(b)) # False

a = [1,2,3] # 원소가 같은 리스트 리터럴이지만 서로 다른 주소를 갖습니다.
b = [1,2,3]
print(id(a)) # 2060084919424
print(id(b)) # 2060084876480

# 파이썬에서 변수 사용하기
# - 리스트, 튜플의 원소를 분해하여 한번에 각각의 변수에 참조 할 수 있습니다.

a,b,c = [1,2,3]
print(a) # 1
print(b) # 2
print(c) # 3

#a,b = [1,2,3] # 분해되어지는 리스트의 원소와 분해받는 변수의 갯수가 일치해야합니다.

a,b = (1,2)
print(a) # 1
print(b) # 2

(a,b) = 'a','b' # 튜플 분해 시 변수에 괄호로 묶을 수도 있습니다.
print(a) # a
print(b) # b

a=b='python' # a와 b 변수에 모두 python 문자열 대입
```


## 제어문
#### if 문 
- 조건이 true 인 경우 실행하는 구간
- 파이썬에서는 구문의 블럭을 들여쓰기로 구분합니다.
- 들여쓰기가 같으면 같은 구문의 블럭이 됩니다.
- 구문 빠져나올 때에는 들여쓰기를 빼주면 됩니다.

```
flag = False

if flag == True:
    print('flag 변수는 True입니다.')
    print('flag 변수가 true입니다.')
    
print('프로그램 종료')
```

## 제어문과 사용할 수 있는 비교, 논리 연산자
- 반환값은 전부 True 또는 False인 bool 타입입니다.

#### 비교연산자
1) 크기비교 연산자
- <. >, <=, >=
- 파이썬에서는 비교연산 체이닝이 가능하여 10 < a < 20 연산식이 가능합니다.
- 10 < a 비교 -> a < 20 비교 -> 1,2,가 모두 True이면 True 반환

```
age = 30

if 20<age<65:
    print('일반성인입니다.')
```

2) 동등비교 연산자
- ==, !=

```
# 파이썬에서 == 연산자는 값의 비교를 합니다.

a = 300
b = 300
print(a == b) # True

# 파이썬에서 is 연산자는 참조 주소의 비교를 합니다.

print(a is b) # True
```

#### 논리 연산자
- 논리곱 (그리고, and) : 피연산자1,2 모두 True -> True
- 논리합 (또는, or) : 피연산자1 또는 피연산자2가 True -> True
- 논리부정 (not) : 피연산자1이 True -> False, False -> True

```
print(True and True) # True
print(True and False) # False

a = 30
print((a>20) and (a<10)) # True and False -> False
print((a>20) or (a<10)) # True and False -> True
```

- 파이썬에서 논리 연산자의 결과는 항상 bool 타입이 아닐수 있습니다.
- Falsy와 Truthy로 연산하는 경우 and 연산은 항상 Falsy 값을 반환합니다.
- Falsy와 Truthy로 연산하는 경우 or 연산은 항상 Truthy 값을 반환합니다.

```
a = []
b = 10
print(a and b) # []

a = ''
b = []
print(a and b) # Falsy and Falsy 인 경우 가장 첫번째(왼쪽) 피연산자의 값을 반환합니다., ' '

a = []
b = 10
print(a or b) # Falsy or Truthy -> Truthy 반환, 10

a = [1,2]
b = 10
print(a or b) # Truthy or Truthy 인 경우 가장 첫번째(왼쪽) 피연산자의 값을 반환합니다., [1,2]

a = []
print(not a) # not은 Truthy -> False, Falsy -> True 로 bool 타입 반환합니다., True
```

#### in, not in 연산자
- 주로 집합 구조 (list,tuple,dict,set) 에서 원소가 있고 없고를 판단할 때 사요하느 연산자입니다.

```
a = ['python', 'java', 'db']

# a 변수 list에 python이 있다면 '현재 python은 존재합니다.'라고 출력하고 싶습니다.
if 'python' in a:
    print('현재 python은 존재합니다.')
    
# a 변수 list에 db가 있다면 '현재 db가 존재합니다.'라고 출력하고 싶습니다.
if 'db' in a:
    print('현재 db가 존재합니다.')
    
# a 변수 list에 java가 없다면 '현재 java는 존재하지 않습니다.'라고 출력하고 싶습니다.
if 'java' not in a:
    print('현재 java는 존재하지 않습니다.')
else:
    print('현재 java가 존재합니다.')
    
# a 변수 list에 javascript가 있다면 '현재 javascript는 존재합니다.'라고 출력하고 싶습니다.
if 'javascript' in a:
    print('현재 javascript는 존재합니다.')
else:
    print('현재 javascript는 존재하지 않습니다.')
```

## 반복문

#### while 문
- 반복의 조건이 True인 동안 계속 반복하는 제어문
- 반복의 조건이 False가 될 수 있도록 실행문을 제어하지 않으면 무한루프에 빠져 시스템이 고장난다.

```
count = 1
while count < 11:
    print(f'{count}번 박복 중')
    count += 1 # 파이썬에서는 증감 연산자 없습니다. (count++ 안됨)
    
count = 1
total = 0
# count 변수를 활용하여 1~30까지의 숫자 중 짝수의 합을 구하여 출력하는 while문을 작성해보세요.
while count < 31:
    if count % 2 == 0:
       total += count # total 변수를 미리 while문 위에 선언하고 초기화 하지 않으면 += 연산자 사용x
    count += 1
print(total)

str = ''
# while문을 사용하여 사용자가 입력한 문자열을 합치는 실행문을 작성해 보세요.
# 만약 사용자가 exit를 입력하면 해당 문자열은 합치지 않고 while문 종료 후 합친 문자열을 출력합니다.
while True:
    text = input('입력> ')
    if text == 'exit':
        break
    str += text
print(str)
```
