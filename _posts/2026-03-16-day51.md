#### 노드 이동 예제
- 이미 DOM에 존재하는 노드를 insertBefore로 다른 위치에 삽입하면 이동이 됨.
- ▲ / ▼ 버튼으로 리스트 항목의 순서를 변경.

```
const $ups = document.querySelectorAll('.up');
$ups.forEach($up => {
    $up.onclick = function(){
        const $li = this.closest('li');
        const $ul = this.closest('ul#exUl');
        if($li == $ul.firstElementChild) return;
        $ul.insertBefore($li, $li.previousElementSibling);
        console.log($li);
        console.log($ul);
    }
});
const $downs = document.querySelectorAll('.down');
$downs.forEach($down => {
    $down.onclick = function(){
        const $li = this.closest('li');
        const $ul = this.closest('ul#exUl');
        if($li == $ul.lastElementChild) return;
        $ul.insertBefore($li.nextElementSibling, $li);                
    }
});
console.log($ups);
```


## 어트리뷰트 노드 (Attribute Node)

#### 어트리뷰트란?
- HTML 요소는 여러 개의 어트리뷰트(속성)를 가질 수 있음.
- 어트리뷰트이름="어트리뷰트값" 형식으로 정의.
- 모든 요소 노드는 Element.prototype.attributes로 취득 가능 (getter 전용, NamedNodeMap 반환).

[ 어트리뷰트 종류 ]

| 종류 | 설명 | 예시 |
| :---: | :--- | :--- |
| **글로벌 어트리뷰트** | 모든 HTML 요소에 공통으로 사용 가능한 속성 | `id, class, style, title, data-*` |
| **이벤트 핸들러 어트리뷰트** | 이벤트 발생 시 실행할 코드를 지정하는 속성 | `onclick, onchange, onfocus, onblur, oninput` |

```
const $ksmart = document.getElementById('ksmart');
console.log($ksmart.attributes);
console.log($ksmart.attributes.id.value);
```

#### HTML 어트리뷰트 조작
- attributes 프로퍼티는 getter 전용이므로 값 변경 불가.
- 값을 읽거나 변경할 때는 요소 노드의 메서드를 사용.

| 메서드 | 설명 | 예시 |
| :---: | :--- | :--- |
| `getAttribute(attr)` | 어트리뷰트 값 반환 | `$el.getAttribute('value')` |
| `setAttribute(attr, val)` | 어트리뷰트 값 설정 | `$el.setAttribute('value', '새값')` |
| `hasAttribute(attr)` | 어트리뷰트 존재 여부 확인 | `$el.hasAttribute('disabled')` |
| `removeAttribute(attr)` | 어트리뷰트 삭제 | `$el.removeAttribute('disabled')` |

```
const $ksStudent = document.getElementById('ksStudent');
$ksStudent.setAttribute('value', '신입개발자');

const $confirmBtn = document.getElementById('confirmBtn');
const $memberId = document.querySelector('input[name="memberId"]');

$confirmBtn.onclick = function(){
    $memberId.setAttribute('value','개발자 58기');
}
const $rsBtn = document.getElementById('rsBtn');
$rsBtn.onclick = function(){
    alert($memberId.getAttribute('value'))
}
```

#### 어트리뷰트 프로퍼티 vs DOM 프로퍼티

[ 어트리뷰트 프로퍼티 vs DOM 프로퍼티 ]

| 구분 | 어트리뷰트 프로퍼티 (getAttribute) | DOM 프로퍼티 (element.value) |
| :---: | :--- | :--- |
| **특징** | HTML의 초기 상태값 유지 | 사용자 입력에 따른 최신 상태 반영 |
| **변경 여부** | 사용자 입력에도 변경되지 않음 | 사용자 입력 시 즉시 변경됨 |
| **사용 예** | 폼 초기화, 기본값 확인 시 | 현재 입력값 읽기/쓰기 |
| **text input** | `getAttribute('value')` &rarr; 초기값 | `element.value` &rarr; 현재 입력값 |
| **checkbox** | `getAttribute('checked')` &rarr; 문자열 또는 null | `element.checked` &rarr; 현재 체크 상태 (boolean) |
| **select** | `getAttribute('selected')` &rarr; 초기 선택 여부 | `element.value` &rarr; 현재 선택된 값 |

```
const $btns = document.querySelectorAll('.btn');
$btns.forEach($btn => {
    $btn.onclick = function(){
        const $formEle = this.previousElementSibling;
        const tagName = $formEle.tagName;
        console.log({$formEle});
        let attrValue = $formEle.getAttribute('value');
        let domValue = $formEle.value;
        
        switch (tagName) {
            case 'INPUT':
                const type = $formEle.getAttribute('type');
                if(type === 'checkbox'){
                    const attr = $formEle.getAttribute('checked');
                    attrValue = attr ?? 'null';
                    domValue = $formEle.checked;
                }
                break;
            case 'SELECT':
                const $options = $formEle.querySelectorAll('option');
                [...$options].some($option => {
                    if($option.getAttribute('selected') !== null){
                        attrValue = $option.getAttribute('value');
                        return true;
                    }
                })
                break;
        }
        alert(`어트리뷰트 값: ${attrValue}, 돔프로퍼티값: ${domValue}`);
    }
});
```

#### data 어트리뷰트와 dataset 프로퍼티
- data-* 어트리뷰트로 HTML 요소에 사용자 정의 데이터를 저장.
- element.dataset으로 취득. data- 이후 이름은 카멜케이스로 변환됨.
- 예: data-goods-name → dataset.goodsName

```
const $addGoodsBtn = document.getElementById('addGoodsBtn');
$addGoodsBtn.onclick = function(){
    console.log($addGoodsBtn.getAttribute('data-seller-id'));
    console.log(this.getAttribute('data-goods-code'));
    const dataset = this.dataset;
    console.log(dataset);
    console.log(dataset.sellerId);
    console.log(dataset.goodsCode);
    console.log(dataset.goodsName);
    console.log(dataset.goodsPrice);
    //sellerId: id001, goodsCode: g001, goodsName: 모니터, goodsPrice: 1000000
    //this.removeAttribute('style');
    /*
    let text = '';
    for(const key in dataset){
        text += `${key}: ${dataset[key]} `;
    }
    */
    this.parentNode.nextElementSibling.removeAttribute('style');
    this.parentNode.nextElementSibling.textContent = Object.keys(dataset)
                                                    .map(key => `${key}: ${dataset[key]}`).join(', ');
}

const createLi = (cate, data) => {
    const $li = document.createElement('li');
    $li.textContent = `${cate}: ${data}`;
    return $li;
}

const $teamBtn = document.getElementById('teamBtn');
$teamBtn.onclick = function(){
    const dataset = this.dataset;
    const copyDS = Object.assign({}, dataset);
    const { teamName } = copyDS;
    const liList = [ createLi('팀이름', teamName) ];
    delete copyDS.teamName;
    //console.log(Object.keys(copyDS));
    Object.keys(copyDS)
            .map(key => copyDS[key])
            .sort()
            .reduce((acc, cur, idx) => {
                acc.push(createLi(`팀원${idx+1}`, cur));
                return acc
            }, liList);
    console.log(liList);
    const domFragment = document.createDocumentFragment();
    domFragment.append(...liList);
    this.parentNode.parentNode.lastElementChild.replaceChildren(domFragment);
}
```
