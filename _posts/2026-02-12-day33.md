## 파이썬
- 1990년 암스테르담의 귀도 반 로섬(Guido Van Rossum)이 개발한 인터프리터 언어
- 구글에서 만든 소프트웨어의 50% 이상이 파이썬으로 작성됨
- 인스타그램(Instagram), 넷플릭스(Netflix), 아마존(Amazon) 등에서 사용
- 공동 작업과 유지 보수가 매우 쉽고 편리함


#### 파이썬 특징
- 파이썬은 인간다운 언어이다
	- 파이썬은 사람이 생각하는 방식을 그대로 표현할 수 있는 언어
- 문법이 쉬워 빠르게 배울 수 있다
	- 문법 자체가 아주 쉽고 간결하며 사람의 사고 체계와 매우 닮아 있음
- 무료이지만 강력하다
	- 오픈 소스이며 무료로 언제 어디서든 파이썬을 다운로드하여 사용 가능
- 간결하다
	프로그램이 실행되게 하려면 줄(들여쓰기)을 반드시 맞추어야 함 가독성 ↑
- 개발 속도가 빠르다

#### 파이썬 활용 
웹 프로그래밍
- 파이썬은 웹 프로그램을 만들기에 매우 적합한 도구
- 파이썬으로 제작한 웹 사이트는 셀 수 없이 많을 정도

인공지능과 머신러닝
- 자연어 처리, 음성 인식, 이미지 인식과 같은 인공지능 기술 구현
- 인공지능과 머신러닝 프로그래밍을 쉽게 할 수 있도록
- scikit-learn, TensorFlow, PyTorch, Keras 등과 같은 다양한 라이브러리 제공

수치 연산 프로그래밍
- C로 작성된 넘파이(Numpy) 수치 연산 모듈을 통해 빠른 수치 연산 가능

데이터 분석
- NumPy, Pandas, Matplotlib 등의 라이브러리를 활용해
- 데이터 처리, 통계 분석, 시각화를 손쉽게 수행
- 판다스가 등장한 이래 데이터 분석에 R보다 파이썬을 사용하는 사례 증가

데이터베이스 프로그래밍
- Sybase, Infomix, Oracle, MySQL, PostgreSQL 등의 데이터베이스에 접근하기 위한 도구 제공
- 자료를 변형 없이 그대로 파일에 저장하고 불러오는 파이썬 모듈 피클(pickle)

시스템 유틸리티 제작하기
- 운영체제(윈도우, 리눅스 등)의 시스템 명령어를 사용하는 도구를 통한 시스템 유틸리티 제작

GUI(Graphic User Interface) 프로그래밍
- 화면에 윈도우 창을 만들고 프로그램을 동작시킬 수 있는
- 메뉴나 버튼, 그림 등을 추가하는 것
- GUI 프로그래밍을 위한 도구들을 갖추고 있어, GUI 프로그램을 만들기 쉬움 (예 - Tkinter(티케이인터))

C/C++와 결합하기
- C나 C++로 만든 프로그램을 파이썬에서, 파이썬으로 만든 프로그램을 C나 C++에서 사용 가능

사물 인터넷
- 라즈베리파이를 제어하며 사물 인터넷 구현

#### 파이썬 활용하기 어려운 일
시스템과 밀접한 프로그래밍 영역
- 운영체제, 엄청난 횟수의 반복과 연산이 필요한 프로그램 등 
- 매우 빠른 속도를 요구하거나 하드웨어를 직접 건드려야 하는 프로그램에는 적합하지 않음

모바일 프로그래밍
- 안드로이드 네이티브 앱(android native app) 개발에는 아직 역부족
- 아이폰 앱 개발은 할 수 없음


## 숫자형
파이썬에서는 숫자형 타입은 리터럴 데이터로 결정됩니다.

#### 정수형

```
a = 10 # 10진수 양의 정수
print(a)
a = -10 # 10진수 음의 정수
print(a)
```

#### 실수형

```
a = 10.5 # 양의 실수
print(a)
a = -10.5 # 음의 실수
print(a)
```

#### 8진수

```
a = 0o12 # 8진수 표현 (0o 또는 0O)
print(a)
```

#### 16진수

```
a = 0xA # 16진수 표현 (0x)
print(a)
```
#### 산술연산자

```
print(10+20) # + 연산자는 덧셈
print(10-20) # - 연산자는 뺄셈
print(10*20) # * 연산자는 곱셈
print(10/20) # / 연산자는 나눗셈
print(10%20) # % 연산자는 너머지
print(10//20) # // 연산자는 몫
print(2**3) # ** 연산자는 제곱
```


## 문자열
-  ''' 또는 """ 를 사용하면 내부 문자열의 줄바꿈, 공백이 유지 됩니다.
- \(이스케이프) 문자를 사용하면 ' 또는 " 를 문자열 표기가 아닌 텍스트로 표현할 수 있습니다.
- 파이썬에는 문단 주석이 없으므로 """ 또는 ''' 를 사용하여 문단 주석을 표현합니다.

```
a = "hello world"
print(a)
a = 'hello world'
print(a)
a = "'hello world'"
print(a)
a = '"hello world"'
print(a)
a = '''hello world'''
print(a)
a = """hello world"""
print(a)

a = '선생님이 \n"python은 쉬워요" \n라고 말했습니다.'
print(a)

# ''' 또는 """ 를 사용하면 내부 문자열의 줄바꿈, 공백이 유지 됩니다.
a = '''
선생님이 
"python은 쉬워요"
라고 말했습니다.
'''
print(a)

# \(이스케이프) 문자를 사용하면 ' 또는 " 를 문자열 표기가 아닌 텍스트로 표현할 수 있습니다.
print('\'hello\'"python"') 
print("'hello'\"python\"") 

# 파이썬에는 문단 주석이 없으므로 """ 또는 ''' 를 사용하여 문단 주석을 표현합니다.

# 문자열 연결하기
print("hello"+", world")

# 문자열 곱하기
print("*"*10)
print("*"*5)
print("*"*4)
print("*"*3)
print("*"*2)
print("*")
```

#### 변수명[인덱스 번호]
- 특정 인덱스의 문자를 반환 합니다.

```
# 문자열은 0부터 인덱스를 가집니다.
a = "Life is too short, You need python"
print(a[0])

# a[0] = 'A' 문자열은 수정이 안됩니다.

print(a[5], a[10])
```

#### 문자열 슬라이싱
- 문자열의 원하는 범위만큼의 문자열을 반환하는 방법

```
b = a[0:4] # 0번 인덱스부터 4번 앞 인덱스까지의 범위 반환
print(b) 
print(a) # 사용한 문자열에 변화는 x

print(a[8:11])

b = a[-6: ] # 마지막 글자까지는 [i: ] 오른쪽을 생략합니다.
print(b) 

b = a[ :4] # 처음부터 슬라이싱을 할 때는 [ :i] 왼쪽을 생략할 수 있습니다.
print(b)

b = a[ : ] # 처음부터 끝까지 모든 문자열 새로 반환 
print(b)
```

## 포매팅

#### 문자열 포매팅
- %d : 정수형
- %s : 문자열형
- %c : 문자형
- %f : 실수형
- %o : 8진수
- %x : 16진수
- %% : '%'문자로 표현 (다른 포매팅 문자열과 사용하는 경우)


```
count = 2
a = "나는 사과를 %d개 먹는다."
print(a %count)

name = "고길동"
count = 5
print("%s님 사과를 %d개 먹었습니다." %(name, count))
```

#### 실수형 포매팅
- 포맷 코드 %f와 숫자를 함께 사용하는 경우 소수점의 자릿수 또는 전체 수의 자릿수를 지정할 수 있습니다.
- %0.4f -> 전체 자릿수 상관 없이 소수점 4자리 표현
- %10.4f -> 전체 자릿수 10자리 중 소수점 4자리 표현 (전체 자릿수가 넘치는 경우 자르지 x)

```
floatNum = 1000000.123456789
print("실수 floatNum-> %f" %floatNum)
print("실수 floatNum-> %0.1f" %floatNum)
floatNum = 100.1
print("실수 floatNum-> %10.4f" %floatNum)
```

#### format 함수를 사용한 포매팅

```
count = 10
name = "고길동"
progress = 80

print("나는 사과를 {0}개 먹는다." .format(count))
print("{0}님 환영합니다." .format(name))
print("오늘 수업 {0}% 진행되었습니다." .format(progress))

# format() 함수 괄호 안의 인자(매개값) 순서대로 0부터 문자열 안에 포맷팅할 수 있습니다.
print("{0} 님이 사과를 {3}개 먹었습니다." .format(name, count, "", 5))

# format() 함수에서 인덱스가 아닌 이름으로 포매팅을 할 수 있습니다.
print("{user} 님이 사과를 {num}개 먹었습니다." .format(user="고길동", num=10))
```
