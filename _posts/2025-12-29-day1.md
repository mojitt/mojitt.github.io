### 웹이란?
- 인터넷을 활용하여 서로 다른 컴퓨터들 끼리 정보를 공유하기 위한 목적의 서비스

### 서버와 클라이언트?
- 서버는 정보를 제공하는 시스템을 탑재한 pc
- 클라이언트는 웹브라우저를 통하여 정보를 요청하는 pc

### URL에 숨어있는 정보는?
ex) http://www.google.com:80/mail/abcdefg
- 프로토콜 : http
- 도메인(ip주소) : www.google.com
- 포트 : 80
- 경로, 요청페이지 : mail/abcdefg

### 브라우저를 통해 url 요청시 응답까지의 과정?
- 클라이언트에서 url을 통하여 웹 페이지 요청
- url속 도메인 사용하여 서버를 찾음 
- 서버쪽에서 클라이언트 요청 승인 
- 클라이언트에서 페이지를 요청 
- 서버쪽에서 해당 포트안의 시스템에서 요청 페이지를 검색 후 찾은 페이지를 클라이언트에 전달 
- 클라이언트 브라우저에서 응답 받은 웹페이지를 해독하여 출력

### 웹페이지 구성요소 3가지
구조(html), 디자인(css), 동작(javascript)

### 헤딩태그의 종류와 특징
h1~h6
- 블럭 라인
- 본문의 제목을 작성할 때 사용
- 굵은 글자
- 1이 제일 크고 6으로 갈수록 작아짐

### 문단 태그의 종류와 특징
p태그
- 블럭 라인
- 블럭 성질의 태그는 중첩할 수 없음

### 연속된 공백과 줄바꿈을 유지하는 태그
pre태그

### 텍스트를 꾸미는 태그의 종류와 특징
- b : 굵게
- strong : 강조
- i : 이탤릭체
- em : 중요표시
- ins : 추가표시
- del : 삭제표시
- sup : 윗첨자
- sub : 아랫첨자
- small : 작게
- mark : 하이라이트
(전부 인라인 성질, 같아 보이지만 컴퓨터가 분류할 때 차이가 있음)

### html에서 블럭과 인라인 영역을 지정하기 위한 태그
- div : 블럭단위로 여러 블럭 또는 인라인인 태그들을 하나의 블럭단위 그룹으로 묶는 태그
- span : 한 라인에서 특정 영역을 쪼갤 때 사용하는 태그

### head 태그 안에 중첩할 수 있는 태그는?
title, base, meta, link, style, script

### 경로를 지정하는 태그와 속성
- base : 기본 경로를 지정하는 태그
- href : 지정하는 태그의 경로값을 작성하는 속성

### 작성하는 웹페이지에 저작권을 표기하고 싶습니다. 태그를 작성해보세요.
&lt;meta name="author" content="홍길동"&gt;

### 이미지를 출력하고 지정하는 태그와 속성
- img : 웹페이지에 원하는 이미지를 출력하는 태그
- src : 삽입하고자 하는 이미지의 경로값을 지정하는 속성
- alt : 이미지가 로드되지 못했을 때 대체 텍스트를 보여주는 속성

### 웹페이지에서 사용할 수 있는 목록의 종류
- 순서가 있는 목록 : ol
- 순서가 없는 목록 : ul
- 정의형 목록 : dl

### 순서가 있는 목록과 없는 목록
순서가 있는 목록
```
<ol>
    <li>리스트</li>
</ol>
```
순서가 없는 목록
```
<ul>
    <li>리스트</li>
</ul>
```

- 글 머리 숫자 타입 변경하기 : type 속성 사용 / 글 머리 숫자 시작번호 변경하기 : start 속성
- 상위 메뉴, 하위 메뉴가 존재하는 경우
- 목록을 정의하는 ul, ol 태그의 자식 태그로는 li 태그 밖에 중첩할수 없으나, li 태그는 항목의 내용을 작성하는 위치로 모든 태그를 선언할수 있다.
- 해당 원리를 사용하여 항목 li 태그 안에 다른 목록 태그를 중첩하여 하위 목록을 작성합니다.
