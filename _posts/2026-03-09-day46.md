## JavaScript 내장 객체 (Built-in Objects)

#### 내장 객체란?
- JavaScript에서 기본적으로 제공하는 객체들로, 다양한 기능을 수행하는 메서드를 포함.
- 주요 내장 객체: Object, String, Number, Boolean, Array, Date, Math, RegExp 등

#### Object 객체
- 모든 JavaScript 객체의 최상위 객체.
- 객체 생성, 복사, 병합, 속성 관리 등의 메서드를 제공.

```
const person = {
    name : '홍길동',
    age : 25,
    city : '전주'
}
console.log(person);

// Object.keys() - 객체가 가지고 있는 키를 배열로 반환
console.log(Object.keys(person));
// Object.values() - 객체가 가지고 있는 값을 배열로 반환
console.log(Object.values(person));
// Object.entries() - 객체가 가지고 있는 [키, 값]을 배열로 반환
console.log(Object.entries(person));
for(const [key,value] of Object.entries(person)){
    console.log(key,value);
}

// Object.assign() - 객체 병합
const info = {
    email : 'hong@ksmart.or.kr',
    age : 30 // 동일 키값을 마지막 합쳐질 객체 속성기준으로 덮어씀
}
const mergeObj = Object.assign({}, person, info);
console.log(mergeObj);

// Object.freeze() - 객체 동결 (수정 불가, 추가 불가)
const statusObj = {
    status : 'active',
}
Object.freeze(statusObj);
statusObj.status = 'stop'; // 수정 불가
statusObj.category = 'payment'; // 추가 불가
console.log(statusObj);

// Object.seal() - 객체 봉인 (수정 가능, 추가 불가)
const sealObj = Object.seal({name : '홍길동'});
sealObj.name = '김철수'; // 수정 가능
sealObj.age = 20; // 추가 불가
console.log((sealObj));

// Object.hasOwn() - 객체 키 여부
console.log(Object.hasOwn(person, 'name'));
console.log('name' in person);
```

#### String 객체
- 문자열을 다루기 위한 메서드를 제공.
- 문자열 검색, 추출, 변환, 분할 등 다양한 작업을 수행.

```

console.log(str.includes('script')); // true, 문자열 포함여부 true/ false
console.log(str.includes('Script')); // false, 대소문자 구분

console.log(str.startsWith('Hello')); // true, 문자열의 시작 글자와 일치시 true, 불일치시 false
console.log(str.startsWith('Javascript')); // false

console.log(str.endsWith('Hello')); // false, 끝나는 문자열과 일치시 true, 불일치시 false
console.log(str.endsWith('Javascript')); // true

console.log(str.charAt(0)); // H, 문자열의 인덱스에 일치하는 문자를 반환
console.log(str.charAt(-1)); // '', -1 일 경우 빈문자열 반환

// 문자열의 유니코드 반환
console.log(str.charCodeAt(0)); // H : 72 반환

// slice(시작인덱스, 종료인덱스미만)

console.log(str.slice(-10)); // Javascript, 음수(문자열 마지막부분부터) 

// substring(처음인덱스포함, 종료인덱스미만)
console.log(str.substring(0,5)); //Hello

// substr(처음인덱스번호, 추출할 갯수)

// split(구분자, 제한갯수)
```

#### Number 객체
- 숫자를 다루기 위한 메서드와 속성을 제공.
- 숫자 변환, 형식화, 유효성 검사 등을 수행.

```
console.log(Number('123')); // 123
console.log(Number('123.12')); // 123.12
console.log(Number('123.123가')); // NaN

// parseInt 정수변환
console.log(parseInt(123.12)); // 123
console.log(parseInt('123.12')); // 123
console.log(parseInt('123px')); // 123

// parseFloat 실수변환
console.log(parseFloat(123)); // 123
console.log(parseFloat('123.123123')); //123.123123

// isNaN true/false
console.log(Number.isNaN(NaN)); // NaN ture
console.log(Number.isNaN(123)); // false
console.log(Number.isNaN('123')); // false
console.log(Number.isNaN('가')); // false

// isInteger
console.log(Number.isInteger(123.123)); // false
console.log(Number.isInteger(123)); // true , 정수만
console.log(Number.isInteger('123')); // false, 문자열 정수
console.log(Number.isInteger('가')); // false

// toFixed
const num = 123.45678;
console.log(num.toFixed(3)); // 123.457 소숫점 세번째자리까지 반올림한 실수 반환

// 나라별 통화 기호에 맞춰 문자열로 반환
console.log((12345678).toLocaleString('ko-KR', {style: "currency", currency: "KRW"}));
```

#### Math 객체
- 수학적 계산을 위한 속성과 메서드를 제공.
- 삼각함수, 로그, 지수, 반올림 등 다양한 수학 연산을 수행할 수 있음.
- 단, Number 자료형만 지원, Bigint 사용할 수 없음.

```
console.log(Math.round(value*100)/100); // 4.57, 소수점 셋째자리에서 반올림

console.log(Math.floor(-4.567*10)/10); // -4.6
console.log(Math.floor(-4.567*100)/100); // -4.57
```

#### Date 객체
- 날짜와 시간을 다루기 위한 메서드를 제공.
- 현재 시간 조회, 날짜 계산, 포맷팅 등을 수행.

```
console.log(now.getDate()); // 일 
console.log(now.getDay()); // 요일 (0(일요일) ~ 6(토요일))
```


#### JSON
JSON (JavaScript Object Notation)
- 데이터를 저장하고 교환하기 위한 텍스트 기반의 경량 데이터 형식.
- JS 객체 문법에서 파생되었지만, 언어에 독립적이라 대부분의 언어에서 사용 가능.
- 클라이언트 ↔ 서버 간 데이터 교환, 웹 스토리지 저장, 설정 파일 등에 폭넓게 활용.

```
const exObj = {
    name : '홍길동',
    fn : function(){},
    obj : {},
    arr : [],
    boolean : true,
    nullKey : null,
    infinity : Infinity
};
console.log(typeof JSON.stringify(exObj));
console.log(JSON.stringify(exObj, null, 2));

const responseJson = JSON.stringify(exObj);
console.log(JSON.parse(responseJson));

// 깊은 복사 (단, 완전한 깊은 복사는 아니다.)
const student = {
    name : '홍길동',
    hobby : [
        '영화감상',
        '음악감상'
    ]
}
const deepCopyStudent = JSON.parse(JSON.stringify(student));
```
