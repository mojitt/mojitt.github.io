## 조건문
#### if 문
특정 조건에 따라 구간을 실행하거나 건너뛰는 문법을 의미합니다.

```
 if(조건식){
실행문...
실행문...
...		 
}
```

조건식이 true이면 블럭 안 실행문들을 위에서부터 아래로 순차적으로 실행 후 원래 코드의 흐름으로 빠져나옵니다. <br>
조건식이 false이면 블럭을 건너뛰고 원래 코드의 흐름으로 실행합니다.

**else 문** 

```
if(조건식) {
실행문1... //조건식 true인 경우 실행
} else {
실행문2... //조건식 false인 경우 실행
}
```

if의 조건이 거짓인 경우 실행되는 구간을 만들 수 있습니다. <br>
필수는 아닙니다. (if 조건식에 대한 거짓 실행이 있는 경우만 작성합니다.)

**else if문**

```
if(조건식1) {
실행문1... //조건식1이 true인 경우 실행
} else if(조건식2) {
실행문2... //조건식1이 false이고 조건식2가 true인 경우 실행
}....
```

if문 또는 윗 조건이 거짓인 경우 다른 조건을 비교하고 만족하는 경우 실행하고 싶은 실행문을 처리하는 구문입니다. <br>
else는 모든 조건이 거짓인 경우 실행문이 있는 경우만 작성합니다.

**다중 if문**

```
if(조건문1){
    if(조건문2){
        실행문1... //조건문1을 만족하고 조건문2도 만족할 때 실행
    } else {
        실행문2... //조건문1을 만족하지만 조건문2가 만족하지 않을 때 실행
    }
    실행문3... //조건문1이 만족할 때 실행문1 또는 실행문2와 함께 출력		
} 
```

if문 안에 또 다른 if문이 중첩되어있는 구조를 의미합니다. <br>
바깥쪽 if문 조건이 true일 때 중첩 if문의 조건을 비교하여 실행합니다.

#### switch 문
식 또는 변수의 값으로 실행문을 분기처리하는 구문입니다.

```
switch(식 또는 변수) {
    case 값1:
        실행문...
        break;
    case 값2:
        실행문...
        break;
        .
        .
        .
    case 값n:
        실행문...
        break;
    default:
        실행문...
}
```

괄호 안 식 또는 변수의 값이 해당하는 case의 실행문을 실행하고 원래 흐름으로 빠져나옵니다. <br>
식 또는 변수의 값은 boolean이 아닙니다. <br>
일치하는 case가 없는 경우 default 실행하고 원래 흐름으로 빠져나옵니다. <br>
일치하는 case부터 최초의 break문까지 아래로 전부 실행하므로 case 실행 끝에서 break문을 작성해줘야 합니다

## 반복문
특정 실행코드들을 반복해서 실행하도록 하는 문법

#### for 문
실행문 반복의 횟수를 알고 있는 경우 사용하는 문법

```
for(초기화식; 조건식; 증감식){
    실행문...
}
```

초기화식으로 반복의 시작점을 정하고 조건식을 실행하여 true이면 중괄호 블럭을 실행합니다. <br>
한 번 반복마다 마지막에 증감식을 실행하여 조건식이 false가 되도록 만들고 종료하여 원래흐름으로 빠져나옵니다.

**다중 for문, 중첩 for문** <br>
for문 안에 for문이 중첩된 문법을 의미합니다.

```
for(초기화식; 조건식; 증감식) {
    for(초기화식; 조건식; 증감식) {
        실행문2...
    }
    실행문1...
}
```

#### while 문
반복의 횟수는 모르나 조건을 알고 있는 경우 사용하는 문법입니다.

```
while(조건식){
    실행문...		 
}
```

조건식이 true인 동안 실행문을 반복합니다. <br>
실행문 블럭에서 while문을 종료할 수 있는 조건을 만들어줘야 합니다. (무한루프방지)

#### do-while 문
조건 상관없이 한번은 무조건 실행해야하는 경우 사용하는 문법

```
do {
    실행문...
} while(조건식);
```

실행문을 실행 후 조건식이 false이면 종료, true이면 반복실행

#### break 문
반복문이 중첩되어 있을 경우 break; 문은 가장 가까운 반복문만 종료 <br>
바깥쪽 반복문까지 종료시키려면 반복문에 이름(라벨)을 붙이고, “break 이름;” 사용

```
for(int count=1; count<=10; count++) {
    for(int count2=1; count2<=10; count2++) { 
        if(count2>5) {
            break; //반복문 중첩시 근접한 반복문 하나만 종료합니다.
        }
        System.out.println(count+"_"+count2);
    }
}

first: for(int count=1; count<=10; count++) { //for문에 first라는 이름 지정
    for(int count2=1; count2<=10; count2++) { 
        if(count2>5) {
            break first; //해당 break로 first 이름인 for문 강제 종료
        }
        System.out.println(count+"_"+count2);
    }
}
```

#### continue 문
반복문의 현재 단계를 중단 <br>
for문, while문, do-while문에서 사용
- for문: 증감식으로 이동
- while문, do-while문: 조건식으로 이동

```
int sum=0;
for(int count=1; count<=10; count++) {
    if(count%2 == 1) {
        continue; //2로 나누었을 때 나머지가 1이 나올 경우 현재 단계를 중단하고 다음 단계로 넘어간다. 홀수 값 계산은 무시하고 짝수 값만 실행
    }
    sum += count;
}
System.out.println(sum);
```
