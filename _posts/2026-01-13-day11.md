## 연산자

#### 부호연산자 (피연산자가 1개로, 단항연산자)
양수, 음수 연산자 <br>
기호 : +, -
```
int v1 = +100;
System.out.println(v1);
System.out.println(-v1); //음수*양수 -> 음수
```

#### 증강연산자 (피연산자가 1개로 단항연산자)
기호 : ++, -- <Br>
1씩 증가 또는 1씩 감소
```
v1 = 10;
		
++v1; //v1 = v1 + 1
System.out.println(v1); //11

--v1; //v1 = v1 - 1
System.out.println(v1); //10
```
- 전위증가연산, 전위감소연산 -> ++피연산자, --피연산자
- 연산의 우선순위 가장 높음
- 후위증가연산, 후위감소연산 -> 피연산자++, 피연산자--
- 연산의 우선수위 가장 낮음

#### 산술연산자
더하기(+), 빼기(-), 곱하기(*), 나누기(/), 나머지(%) <br>
사칙연산 그대로 곱하기, 나누기의 우선순위가 높고 더하기 빼기 우선순위가 낮다.
```
int num1 = 10;
int num2 = 3;

int result;

result = num1 + num2;
System.out.println("더하기: "+result);

result = num1 - num2;
System.out.println("빼기: "+result);

result = num1 * num2;
System.out.println("곱하기: "+result);

result2 = (double) num1 / num2; // 실수 연산을 위하여 피연산자 하나를 double 강제 형변환
System.out.println(result2);

result = num1 % num2; // 나머지 연산
System.out.println(result);
```

#### 논리연산자
피연산자가 논리타입인 boolean 이여야 합니다.

1. 논리곱(AND 연산자) <br>
피연산자1이 true 이고 그리고 피연산자2가 true 일 때만 true인 연산자 <br>
연산기호 : &&, &
    
\* 피연산자1이 false이더라도 피연산자2를 실행하고 결과 도출 <br>
true  & false -> false <br>
true  & true  -> true <br>
false & false -> false <br>
false & true  -> false

\* 피연산자1이 false이면 피연산자2를 실행하지 않고 결과 도출 (성능면에서 더 많이 사용) <br>
true  && false -> false <br>
true  && true  -> true <br>
false && false -> false <br>
false && true  -> false

```
System.out.println(true && false); // false
System.out.println(true && true); // true
System.out.println(false && false); // false
System.out.println(false && true); // false
```

2. 논리합(OR 연산자) <br>
연산자 : || 또는 |

\* 피연산자1이 true이더라도 피연산자2를 실행하여 결과 도출 <br>
true  | false -> true <br>
true  | true  -> true <br>
false | false -> false <br>
false | true  -> true

\* 피연산자1이 true이면 피연산자2를 실행하지 않고 결과 도출 (성능면에서 많이 사용) <br>
true  || false -> true <br>
true  || true  -> true <br>
false || false -> false <br>
false || true  -> true

```
System.out.println(true || false); // true
System.out.println(true || true); // true
System.out.println(false || false); // false
System.out.println(false || true); // true
```

3. 베타적 논리합 (XOR 연산자) <br>
피연산자1과 피연산자2의 논리값이 서로 다르면 true 서로 같으면 false <br>
연산자 : ^

true  ^ false -> true <br>
true  ^ true  -> false <br>
false ^ false -> false <br>
false ^ true  -> true

```
System.out.println(true ^ false); //true
System.out.println(true ^ true); //false
System.out.println(false ^ false); //false
System.out.println(false ^ true); //true
```

4. 논리부정 (NOT 연산자)  <br>
논리값이 true -> false, false -> true 값 도출 <br>
연산자 : !

```
System.out.println(!true); //false
System.out.println(!false); //true
```

#### 대입연산자

1. 단순 대입 연산자 <br>
연산자 : =, 피연산자2 값을 피연산자1(변수)에 대입 <br>
연산의 방향 : 오른쪽에서 왼쪽

2. 복합 대입 연산자 <br>
+=(합하여 대입), -=(차감하여 대입), *=(곱하여 대입), /=(나누어 대입), %=(나머지를 구하여 대입)

```
int num = 10; //단순 대입 연산자

int num2 = 10;
int num3 = 20;

num += 10; // num = num + 10
System.out.println(num); // 20

num -= 10; // num = num - 10
System.out.println(num); // 10

num *= 10; // num = num * 10
System.out.println(num); // 100

num /= 30; // num = num / 30
System.out.println(num); // 3

num %= 2; // num = num % 2
System.out.println(num); // 1
```

#### 문자열 연산자
피연산자1(문자열)과 피연산자2(문자열)을 합친 문자열을 반환합니다. <br>
연산기호 : +

```
System.out.println("10"+20); // "10"+"20" -> "1020"
		
System.out.println("sum : "+10+20); // sum : 1020
System.out.println("sum : "+(10+20)); // sum : 30
```

\+ 연산자는 산술의 덧셈 연산자이기도 하기 때문에 문자열 연산자와 사용시 주의해야합니다. <br>
만약 피연산자 중 하나라도 문자열 타입이 있다면 문자열로 자동형 변환되어 + 문자열 합치기 연산이 됩니다. <br>
우선순위도 덧셈과 동일하기 때문에 왼쪽에서 오른쪽으로 연산을 처리합니다. <br>
문자열타입 + 숫자형타입 + 숫자형타입 -> 왼쪽에서부터 문자열 합치기로 연산의 결과를 도출 <br>
문자열타입 + (숫자형타입 + 숫자형타입) -> 산술 덧셈 후 왼쪽에서부터 문자열 합치기로 연산의 결과를 도출 


#### 조건연산자 (삼항 연산자)
문법) (피연산자1)?피연산자2:피연산자3
- 피연산자1이 true이면 피연산자2 값 도출, false이면 피연산자3 값 도출

```
int result = (true)?10 : 20;
System.out.println(result); // 10
```
