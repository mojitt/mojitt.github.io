### 글꼴(폰트) 관련 속성
4.front-style(서체)
```
<head>
    <style>
        .p01:hover{font-style: italic;}
        .p02:hover{font-style: normal;}
    </style>
</head>
<bdoy>
    <p class="p01">italic 체로 글자를 옆으로 기울이는 영역</p>
    <p class="p02">normal 체로 글자를 바르게 출력하는 영역</p>
</bdoy>
```
- 특정 영역의 글자를 옆으로 기울이거나 바르게 출력하는 속성입니다.
- *외부글 인용시 주로 사용
        
사용할 수 있는 속성값 종류)
1. italic
2. normal

5.font-variant(작은대문자)
```
<head>
    <style>
        .p01:hover{font-variant: small-caps;}
        .p02:hover{font-variant: normal;}
    </style>
</head>
<bdoy>
    <p class="p01">small-caps를 이용하여 작은 대문자 변경 영역 / ABC</p>
    <p class="p02">normal을 이용하여 원래 소문자로 변경 영역 / ABC</p>
</bdoy>
```
- 특정 영역 안에 있는 영문 소문자를 작은 대문자로 변경하는 속성입니다.
- *대문자에는 영향을 미치지 않는다.

사용할 수 있는 속성값 종류)
1. small-caps
2. normal

6.line-height(줄간격)
```
<head>
    <style>
        .p01{line-height: 0.8em;}
    </style>
</head>
<body>
    <div class="p01" style="border: 1px solid black; font-size: 20px;">
        줄간격을 확인해볼 태그 영역입니다. <br>
        줄간격을 확인해볼 태그 영역입니다. <br>
    </div>
</body>
```
- 특정 영역 안에 줄간격을 조절할 수 있는 속성입니다.
- *한글자 크기보다 작게 값을 적용하면 줄간격이 좁아져 겹쳐질 수 있습니다.
- *한글자 크기보다 크게 값을 적용하면 줄간격이 넓어집니다.
- *line-height 속성은 내용을 세로 중앙에 위아래 균일한 여백을 갖도록 정렬하므로 태그의 높이와 line-height 속성의 높이가 같으면 세로 중앙 배치를 할 수 있습니다. (내용이 한 줄인 경우에만)

주로 사용되어지는 속성값 단위)
1. px <br>
2. em <br>
3. rem <br>

7.font(단축 속성)
```

        .p01{font: italic bold small-caps 1em/1.3em '궁서체', cursive}
    </style>
```

- 폰트 관련 6가지 속성을 한번에 작성할 수 있는 단축 속성입니다.

속성값 작성 순서)
- font-style font-weight font-variant font-size/line-height font-family 순으로 작성
- *필수 작성 요소 : font-size, font-family

8.웹폰트(외부 폰트 적용)
- 타 서버에 있는 외부 폰트를 나의 웹페이지에 적용하는 방법입니다.

적용 방법)
1.다운로드 방법
```
<head>
    <style>
        @font-face{
            src: url('Nanum_Pen_Script/NanumPenScript-Regular.ttf') format(truetype);
            /*폰트 형식이 ttf -> format(truetype) / woff -> format(woff)*/
            font-family: 'nanum pen';
        }
        .p01{font-family: 'nanum pen', cursive;}
    </style>
</head>
<body>
    <p class="p01">
        다운로드 방법을 통하여 웹 폰트를 적용할 영역입니다.
    </p>
</body>
```

2.링크형식 방법 <br>
link 방식
```
<head>
    <link rel="preconnect" href="https://fonts.googleapis.com">
    <link rel="preconnect" href="https://fonts.gstatic.com" crossorigin>
    <link href="https://fonts.googleapis.com/css2?family=Black+Han+Sans&display=swap" rel="stylesheet">
    <style>
        .web .p03{
            font-family: "Black Han Sans", sans-serif;
            font-weight: 400;
            font-style: normal;
        }
    </style>
</head>
<bdoy>
    <p class="p03">
        링크 방식을 통하여 웹 폰트를 적용할 영역입니다.
    </p>
</bdoy>
```
import 방식
```
<head>
    <style>
        @import url('https://fonts.googleapis.com/css2?family=Hi+Melody&display=swap');
        .web .p04{
            font-family: "Hi Melody", sans-serif;
            font-weight: 400;
            font-style: normal;
        }
    </style>
</head>
<bdoy>
    <p class="p04">
        링크 방식을 통하여 웹 폰트를 적용할 영역입니다.
    </p>
</bdoy>
```

### 텍스트 관련 속성2
4.letter-spacing(글자간 간격 조절)
```
<head>
    <style>
        .p01{letter-spacing: -5px;}
    </style>
</head>
<bdoy>

```
- 특정 영역 안에 한 글자 단위 간격을 조정할 수 있는 속성입니다.
- *글자와 글자 사이에 대한 수치를 의미하며 양수이면 넓어지고 음수이면 좁아집니다.

사용하는 주요 단위) 
1. px 
2. em 
3. rem

5.word-spacing(띄어쓰기 간격 조절)

- 단어 사이 간격을 조절할 수 있는 속성입니다.
- 단어 구분은 공백을 의미한다. 띄어쓰기 기준 사이 간격을 의미합니다.
- *띄어쓰기의 간격을 조절하며 양수이면 넓어지고 음수이면 좁아집니다.

사용하는 주요 단위)
1. px
2. em 
3. rem

6.text-transform(영어 대소문자 제어)
```
<head>
    <style>
        .p01{text-transform: uppercase;}
        .p02{text-transform: lowercase;}
        .p03{text-transform: capitalize;}
        .p04{text-transform: capitalize;}
    </style>
</head>
<bdoy>
    <p class="p01">apple banana orange</p>
    <p class="p02">APPLE BANANA ORANGE</p>
    <p class="p03">apple banana orange</p>
    <p class="p04">APPLE BANANA ORANGE</p>
</bdoy>
```
- 특정 영역의 영문을 대소문자 제어할 때 사용하는 속성입니다.
- *capitalize 속성 값은 어절 첫글자가 소문자인 경우에만 대문자로 변경하며 다른 문자들이 대소문자인 것은 변경하지 않는다.

사용할 수 있는 속성값들)
1. uppercase : 전부 대문자로
2. lowercase : 전부 소문자로
3. capitalize : 어절 첫글자 대문자

7.text-shadow(글자 그림자)
```
<head>
    <style>
        .p01{text-shadow: 10px 10px 5px skyblue;}
    </style>
</head>
<bdoy>
    <p class="p01">텍스트 그림자 예제</p>
</bdoy>
```
- 특정 영역의 텍스트에 그림자를 적용할 수 있는 속성입니다.

속성값 나열 순서) <br>
- (수평 위치) (수직 위치) (흐림/번짐 크기) (색상)
- *수평, 수직 값은 필수값 입니다.
- *색상을 생략하면 그림자 색상은 글자 색을 따라갑니다.
- *수평값이 양수이면 그림자 오른쪽으로 이동합니다.
- *수직값은 양수이면 그림자 아래로 이동합니다.

### 컨텐츠 박스 영역 속성
1.width (너비)
```
<head>
    <style>
        div, span{border: 1px solid black;}
        .ex01{width: 400px;}
        .ex02{width: 50%;}
        .ex03{width: 50vw; min-width: 500px; max-width: 600px;}
    </style>
</head>
<bdoy>
    <div class="ex01">px</div>
    <span class="ex01">px</span> <br>
    <div class="ex02">%</div>
    <div class="ex03">
        vw<br>
        <table border="1" width="500">
            <tbody>
                <tr>
                    <td>1</td>
                    <td>2</td>
                    <td>3</td>
                </tr>
            </tbody>
        </table>
    </div>
</bdoy>
```
- 컨텐츠 영역의 너비 값을 제어하는 속성입니다.
- 인라인 성질의 태그에는 width를 변경할 수 없습니다.
- 인라인 블럭 성질의 태그에는 width를 적용할 수 있습니다.(img video, iframe 태그 등)
- *min-width 속성을 사용하여 상대적 너비값을 적용했을 때 최소 너비를 지정할 수 있습니다. <br>
- *max-width 속성을 사용하여 상대적 너비값을 적용했을 때 최대 너비를 지정할 수 있습니다. <br>

주로 사용하는 단위)
1. px : 고정길이로 브라우저 창 너비 상관없이 언제나 px 단위 값을 고정하여 너비를 갖는다.
2. % : 부모 너비에 대한 상대적이 값으로 만약 부모에게 너비가 없다면 브라우저 너비 대비 퍼센트 값이 된다.
3. vw : 웹페이지를 보이고 있는 창을 기준으로 100vw는 100%로 창에대한 상대적인 수치로 너비를 갖는다.

2.heigth (높이)
```
<head>
    <style>
        div{border: 1px solid black;}
        .ex01{height: 100px;}
        .ex02{height: 50%;}
        .ex03{height: 50vh; min-height: 300px; max-height: 400px;}
    </style>
</head>
<bdoy>
    <div class="ex01">px</div>
    <div class="ex02">%</div>
    <div class="ex03">
        vh
        <table border="1">
            <tbody>
                <tr height="100">
                    <td>1</td>
                </tr>
                <tr height="100">
                    <td>1</td>
                </tr>
                <tr height="100">

                </tr>
            </tbody>
        </table>
    </div>

```
- 컨텐츠 영역의 높이를 제어하는 속성입니다.
- *min-height 속성을 사용하여 최소 높이값을 지정할 수 있습니다. 
- *max-height 속성을 사용하여 최대 높이값을 지정할 수 있습니다.

사용할 수 있는 단위)
1. px : 고정길이로 정해진 px 길이만큼 높이를 적용합니다.(창의 높이 영향x)
2. % : 부모태그에 높이가 없다면 브라우저에서 높이는 고정이 아니므로 높이 없음으로 바라보기 때문에 %단위 정용x 
3. vh : 웹페이지를 보이고 있는 창의 높이 기준에 대한 상대적인 수치로 적용합니다.

### 테두리 박스 영역 속성

### 패딩 박스 영역 속성
패딩(padding)
- 패딩이란, 컨텐츠와 테두리 사이 공간으로 내부 여백을 제어할 수 있는 영역을 의미합니다. <br>
- 배경색상은 패딩+컨텐츠 영역에 전부 적용이 됩니다.

```
<head>
    <style>
        .ex02 .box1{
            width: 80px;
            border: 1px solid black;
            padding-top: 20px;
            background-color: skyblue;
        }
        .ex02 .box2{
            width: 80px;
            border: 1px solid black;
            padding-bottom: 20px;
        }
        .ex02 .box3, .ex02 .box4{
            border: 1px solid black;
            width: 80px;
        }
        .ex02 .box3{
            padding-right: 20px;
        }
        .ex02 .box4{
            padding-left: 20px;
        }
    </style>
</head>
<bdoy>
    <div class="ex02">
        <h3>상단 내부 여백</h3>
        padding-top 속성 사용
        <div class="box1">padding</div>

        <h3>하단 내부 여백</h3>
        padding-bottom 속성 사용
        <div class="box2">padding</div>

        <h3>오른쪽 내부 여백</h3>
        padding-right 속성 사용
        <div class="box3">padding</div>

        <h3>왼쪽 내부 여백</h3>
        padding-left 속성 사용
        <div class="box4">padding</div>
    </div>
</bdoy>
```
padding (단축 프로퍼티)
- padding 단축 프로퍼티를 사용하면 내부 여백 4면에 대한 값을 한번에 작성할 수 있습니다.
- 문법) padding: 위여백수치 오른쪽여백수치 아래여백수치 왼쪽여백수치
- 수치를 3개 작성하는 경우 : 위, 아래 다르고 좌, 우 같은 경우 ex) padding: 5px 10px 10px; 
- 수치를 2개 작성하는 경우 : 위, 아래 같고 좌, 우 같은 경우 ex) padding: 10px 20px;
- 수치를 1개 작성하는 경우 : 위, 아래, 좌, 우 모두 같은 경우 ex) padding: 10px;

### 마진 박스 영역 속성
마진(margin)
- 마진아린, 테두리 바깥쪽 영역으로 주로 태그와 태그 사이 간격을 조절하는 박스입니다.
- 아래 속성들을 사용하여 테두리 바깥 4면의 여백을 각각 제어할 수 있습니다.

1. margin-top
2. margin-right 
3. margin-bottom
4. margin-left
- *테두리 바깥쪽으로 배경의 영향을 받지 않습니다. (부모의 배경을 투사합니다)
