### 박스 모서리 둥글게 만들기
**border-radius**
```
<head>
    <style>
        .box1{
            width: 200px;
            height: 100px;
            border: 2px solid black;
            border-radius: 100px;
        }
    </style>
</head>
<body>
    <div class="box1"></div>
</body>
```
- 각 모서리에 원을 위치하고 면과 면이 만나는 지점까지의 각을 지워 둥근 모서리를 만드는 속성입니다.

### 배경을 다루는 속성들
**1.background-color (배경색)**
```
<head>
    <style>
        .box1{
            background-color: rgba(255, 0, 0, 0.5);
            width: 100px;
            height: 100px;
        }
    </style>
</head>
<body>
    <div class="box1">text</div>
</body>
```
- padding, content 영역에 색상을 지정합니다.
- 속성값으로는 색상명, rgb, rgba, hex(16진수 코드)등 가능

**2.background-image (배경이미지)**
```
<head>
    <style>
        .box1{
            width: 300px;
            height: 300px;
            background-image: url(하늘.jpg);
        }
    </style>
</head>
<body>
    <div class="box1"></div>
</body>
```
- padding, content 영역에 배경 이미지를 지정합니다.

**3.background-repeat (이미지반복)**
```
<head>
    <style>
        .box1{
            width: 300px;
            height: 300px;
            background-image: url(하늘.jpg);
            background-repeat: no-repeat;
            border: 1px solid black;
            background-color: skyblue;
        }
        .box2{
            width: 300px;
            height: 300px;
            margin-top: 10px;
            background-image: url(하늘.jpg);
            background-repeat: repeat-x;
            border: 1px solid black;
            background-color: skyblue;
        }
        .box3{
            width: 300px;
            height: 300px;
            margin-top: 10px;
            background-image: url(하늘.jpg);
            background-repeat: repeat-y;
            border: 1px solid black;
            background-color: skyblue;
        }
        .box4{
            width: 300px;
            height: 300px;
            margin-top: 10px;
            background-image: url(하늘.jpg);
            background-repeat: repeat;
            border: 1px solid black;
            background-color: skyblue;
        }
    </style>
</head>
<body>
    <div class="box1"><h3 style="color: white;">no-repeat</h3></div>  
    <div class="box2"><h3 style="color: white;">repeat-x</h3></div>      
    <div class="box3"><h3 style="color: white;">repeat-y</h3></div>      
    <div class="box4"><h3 style="color: white;">repeat(기본값)</h3></div> 
</body>
```
- 영역 안 이미지들의 반복 여부를 지정합니다.
1. no-repeat : 반복 안함
2. repeat-x : 가로로 반복
3. repeat-y : 세로로 반복 
4. repeat : 가로/세로 반복 

**4.background-position (배경이미지 위치)**
```
<head>
    <style>
        div{
            width: 300px;
            height: 300px;
            border: 1px solid black;
            background-color: skyblue;
        }
        .box1{
            background-image: url(하늘.jpg);
            background-repeat: no-repeat;
            background-position: center top;
        }
    </style>
</head>
<body>
    <div class="box1"></div>
</body>
```
- 배경 이미지의 출력 위치 지정할 수 있는 속성입니다.
- x축 : left, right, center
- y축 : top, center, bottom 

문법) background-position : x축 y축
- y축은 생략시 center로 고정됩니다.

**5.bakcground-size (배경이미지 크기)**
```
<head>
    <style>
        .box1{
            background-image: url(하늘.jpg);
            margin-bottom: 5px;
            background-size: 300px 100px;
        }
        .box2{
            background-image: url(하늘.jpg);
            margin-bottom: 5px;
            background-size: 70% 30%;
        }
        .box3{
            background-image: url(하늘.jpg);
            margin-bottom: 5px;
            background-size: cover;
        }
    </style>
</head>
<body>
    <div class="box1" style="color: white;">px</div>
    <div class="box2" style="color: white;">%</div>
    <div class="box3" style="color: white;">cover</div>
</body>
```
- 배경이미지의 크기를 조절하는 속성입니다. 
        
사용할 수 있는 단위)
1. px : 고정크기로 가로 너비만 작성한 경우 이미지 배율에 맞춰 세로 자동 조절됩니다. <br>
문법) background-size: 가로px [세로px]; 
2. % : 배경이미지를 출력하는 태그의 크기에 대한 백분율로 맞춰 배경이미지 크기를 조절합니다. <br>
문법) background-size: 가로% [세로%]; 
3. cover : 배경이미지를 출력하는 태그의 가로, 세로 크기에 맞춰 이미지 크기를 덮어 출력합니다.

**6.background-clip**
```
<head>
    <style>
        .box1{
            background-color: skyblue;
            width: 200px;

            /*border: 10px solid black;*/

```
- 배경을 적용하는 박스를 지정할 수 있는 속성입니다.

속성값 종류)
1. border-box : 테두리 영역까지 배경적용
2. padding-box : 패딩 영역까지 배경적용(기본값)
3. content-box : 컨첸츠 영역까지 배경적용


**7.backgorund 단축 프로퍼티**
```
<body>
    <div style="width: 100px; height: 100px; background: skyblue url(하늘.jpg) no-repeat;"></div>

```
- 6개의 배경 속성을 한번에 정의하는 단축 속성을 의미합니다. 

작성순서) 
- color image repeat position/size clip 
- 전부 생략 가능하며 size를 정의할 때는 position/ 가 필요합니다.


### 박스 그림자 속성
**box-shadow**
```
<head>
    <style>
        .box1{
            width: 100px;
            height: 100px;
            border: 1px solid black;
            box-shadow: 10px 10px 2px 10px gray;
            background-color: blue;
        }
    </style>
</head>
<body>
    <div class="box1"></div>
</body>
```
- 출력된 태그 상자에 그림자를 적용하는 속성입니다.

속성값 나열 순서)
- 수평수치 수직수치 흐림수치 번짐수치 색상 [inset]
- *수평, 수직 수치는 생략 할 수 없습니다.
- *흐림은 박스의 크기 안에서 흐림효과를 수치만큼 적용합니다.
- *번짐은 박스의 크기보다 4면의 그림자 크기를 수치만큼 확대 하여 적용합니다.
- &inset 사용시 테두리 안쪽 그림자를 적용할 수 있으며, 이때 음수, 양수는 반대로 적용합니다.

3. inline-block : 해당 태그들의 성질을 모두 인라인블럭으로 변경합니다.
- *인라인 블럭 성질은 너비, 높이, margin 위아래도 모두 적용되며 옆으로 쌓이는 성질의 태그입니다.

### position 속성
**position 속성**
```
<head>
    <style>
        .box1 div{
            width: 50px;
            height: 50px;
            background-color: yellow;
            border: 1px solid black;
        }
        .box1 .here{
            position: relative;
            left: 51px;
            top: 52px;
        }

        .box2{
            border-top: 1px solid black;
            margin-top: 10px;
            padding-top: 10px;
        }
        .box2 div{
            width: 50px;
            height: 50px;
            background-color: yellow;
            border: 1px solid black;
            display: inline-block;
        }
        .box2 .here{
            background-color: orangered;
        }
        .box2:active .here{
            position: relative;
            left: 56px;
        }

        .box3{
            border-top: 1px solid black;
            padding-top: 10px;
            margin-top: 10px;
            position: relative;
        }
        .box3 div{
            width: 100px;
            height: 100px;
            background-color: khaki;
            position: absolute;
            top: 10px;
            left: 150px;
        }
        .box5{
            margin-top: 100px;
            padding-top: 20px;
            border-top: 1px solid black;
        }
        .box5 div{
            width: 100px;
            height: 100px;
            background-color: brown;
            position: fixed;
            bottom: 10px;
            right: 10px;
        }
    </style>
</head>
<body>
    <div class="box1">
        relative 확인용
        <div>1</div>
        <div class="here">2</div>
        <div>3</div>
    </div>
    <div class="box2">
        <div>1</div>
        <div>2</div>
        <div class="here">3</div>
        <div>4</div>
        <div>5</div>
    </div>

    <div class="box3">
        absolute 확인용
        <div></div>
    </div>
    <div class="box5">
        <div></div>
    </div>
</body>
```
- 평면에 위치하는 요소(태그)들을 층을 분리하여 한층 위에서 원하는 위치로 배치하는 속성입니다.

속성값 종류) 
1. static : 기본값으로 평면의 제 위치를 의미합니다.

2. relative : 현재위치 기준 이동을 의미합니다. 
- position: relative 선언 시 층이 분리되며 이동할 준비를 마칩니다. 그 후 top, left, right, bottom 속성들을 사용하여 현재 위치 기준 이동할 수치를 선언합니다.
- *기존 위치는 사라지지 않고 보존한 상태로 자신의 위치만 이동합니다. 

3. absolute : 부모위치 기준 이동을 의미합니다.
- position: absolute 선언 시 이동할 준비를 마칩니다. 그 후 top, left, right, bottom 속성들을 사용하여 부모 위치 기준 이동할 수치를 선언합니다.
- *부모 층이 분리되지 않았다면 (부모에게 position 선언이 안되었다면) 현재 창 4면을 기준으로 배치합니다.

4. fixed : 창의 위치 기준 이동을 의미합니다.
- 현재 웹페이지 브라우저 창을 기준으로 이동합니다.
- position: fixed 를 선언하면 층이 분리되어 창 기준 이동할 준비를 합니다. 그 후 top, left, right, bottom 속성들을 사용하여 창의 위치 기준 이동할 수치를 선언합니다. 

- top -> 위에서에 대한 기준 (양수/아래로, 음수/위로)
- left -> 왼쪽에서에 대한 기준 (양수/오른쪽, 음수/왼쪽)
- right -> 오른쪽에서에 대한 기준 (양수/왼쪽, 음수/오른쪽)
- bottom -> 아래에서에 대한 기준 (양수/위로, 음수/아래로)

**z-index 속성**
- 요소들을 층 분리하여 서로 겹쳐진 경우 더 윗 층으로 올릴 태그를 지정할 때 사용하는 속성입니다.
- 기본값은 0을 가지고 있으며 정수를 속성값으로 작성합니다. 
- 더 큰 값을 갖는 요소(태그)가 위로 올라옵니다. 
- *평면일 때는 적용되지 않으며 position 혹은 float 속성을 사용하여 층이 분리된 경우에만 사용 가능합니다.

### float 속성
**float 속성**
```
<head>
    <style>
        .box1{
            border: 1px solid black;
            padding: 20px;
        }
        .box1 div{
            width: 100px;
            height: 100px;
            border: 1px solid orange;
            float: left;
        }
        .box1 p{
            border: 2px solid green;
        }
    </style>
</head>
<body>
    <div class="box1">
        <div></div>
        <p>
            가나다라마바사아자차카타파하
            가나다라마바사아자차카타파하
            가나다라마바사아자차카타파하
        </p>

        <p style="clear: left;">   
            가나다라마바사아자차카타파하
            가나다라마바사아자차카타파하
            가나다라마바사아자차카타파하
            가나다라마바사아자차카타파하
        </p>
    </div>
</body>
```
- 워드, 한글의 이미지 어울림 효과 처럼 요소 주변의 텍스트 들을 높이에 맞춰 정렬해주는 속성입니다.
- *float이 적용된 요소가 층 위로 올라가면 아래 있는 요소가 float 적용된 요소의 위치로 올라와 겹쳐지지만 텍스트 등 내부 내용은 겹치지 않게 옆으로 밀어서 보여지는 원리입니다.
- *float이 적용된 요소의 높이 만큼 아래 있는 요소들이 전부 올라옵니다. 이때 가려진 모든 내용을 옆으로 밀려 출력됩니다.

속성값 종류) 
1. left
2. right

**clear 속성**
- clear 속성을 사용하면 float 때문에 영향을 받는 태그를 제외할 수 있습니다.

속성값 종류) 
1. left : 왼쪽 float 제거 
2. right : 오른쪽 float 제거 
3. both : 양쪽 float 제거 

### visibility 속성
**visibility 속성**
```
<head>
    <style>
        .box{
            padding: 20px;
            border: 1px solid black;
        }
        .box div{
            width: 50px;
            height: 50px;
            background-color: yellow;
            margin: 5px 0px;
        }
        .hidden{
            visibility: hidden;
        }
        .box:hover .hidden{
            visibility: visible;
        }
        .none{
            display: none;
        }
        .box:hover .none{
            display: block;
        }
    </style>
</head>
<body>
    <div class="box">
        <div>1</div>
        <div class="hidden">2</div>
        <div class="none">3</div>
        <div>4</div>
    </div>
</body>
```
- 특정 위치 영역은 남기고 숨기거나 보이게 하는 속성입니다.

사용할 수 있는 속성값) 
1. hidden : 위치는 남기고 숨김처리를 합니다. 
2. visible : 해당 위치에 나타납니다.

- *display: none 사용하면 위치도 함께 숨길수 있습니다.

```
                <td>2</td>

            </tr>
            <tr>
                <td>4</td>
                <td>5</td>
                <td>6</td>
            </tr>
        </tbody>
    </table>
</body>
```
- 이 중으로 표현되는 테이블 테두리를 합치거나 다시 이중으로 출력할 때 사용하는 속성입니다. 
        
사용할 수 있는 속성값) 
1. collapse : 한 선으로 합쳐서 테두리 출력합니다.
- *collapse 일때는 tr에 border를 사용하여 행단위 테두리를 제어할 수 있습니다. 
2. seperate : 이중선으로 분리하여 테두리 출력합니다. 

**2. border-spacing**
```
<head>
    <style>
        table{
            border-collapse: collapse;
            border-spacing: 20px;
        }
    </style>
</head>
<body>
    <table border="1">
        <tbody>
            <tr>
                <td>1</td>
                <td>2</td>
                <td>3</td>
            </tr>
            <tr>
                <td>4</td>
                <td>5</td>
                <td>6</td>
            </tr>
        </tbody>
    </table>
</body>
```
- 테이블 테두리와 셀 테두리 간의 간격을 조절하는 속성입니다.
- *테이블 테두리가 seperate로 분리된 상태일때만 사용 가능한 속성입니다.
