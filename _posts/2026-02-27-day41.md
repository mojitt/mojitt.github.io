#### 반환값이 있는 함수

```
# 함수가 튜플을 반환하는 경우
# - 함수의 호출 결과값은 늘 한개 입니다. (반환값은 항상 1개)
# - 튜플을 사용하면 여러개를 반환하는 것처럼 사용할 수 있습니다.

def func1():
    return('a','b')

result = func1() # result = ('a','b')
print(result)

result1, result2 = func1() # result
print(result1)
print(result2)
```

```
def func3():
    return 'a'
    return 10

result = func3()
print(result) # 파이썬에서 함수는 return문을 여러번 작성해도 오류가 발생하지 않지만 첫번째 return문의 값을 반환합니다.
```

```
def func4():
    print('반환값 없음')
    
result = func4() # 파이썬 함수는 return이 없으면 'None'을 반환합니다.
print(result)
```

```
# 매개변수에 초기값을 설정하기

def func5(arg1,arg2,arg3='c'):
    print(arg1)
    print(arg2)
    print(arg3)

func5('a','b','c')
func5('a','b') # 3번째 인수가 없으므로 arg3 매개변수는 'c'로 초기화
func5('a','b','d') # 3번째 인수가 있으므로 arg3 매개변수는 'd'로 할당

def func6(arg1,arg2,*arg3):
    print(arg1)
    print(arg2)
    for e in arg3:
        print(e)
    
func6('a','b','c')
func6('a','b','c','d','e')
```

#### lambda 함수
- return이 존재하는 간단한 함수를 정의하는 예약어 입니다.
- lambda 함수는 이름이 없는 일회성 함수를 정의하는 방법입니다.
- 파이썬에서는 함수 역시 객체로 구분합니다.
- 정의된 lambda 함수는 반드시 변수에 참조되어 사용할 수 있습니다.

```
def func7(num1,num2):
    return num1+num2

result = func7(1,2)
print(result)

sum = lambda num1, num2: num1+num2

result = sum(3,5)
print(result)

def func8(funcArg):
    result = funcArg() # 매개변수 funcArg에 함수를 할당 받아 호출합니다.
    print(result)

#func8(10) # func8 함수는 호출시 매개값으로 반드시 함수를 전달해야합니다.
func8(lambda : 'a')
```

```
# 두 문자열을 사용하여 더 길거나 더 짧은 문자열을 검색하는 함수 정의
str1 = 'abc'
str2 = 'abcdef'

def checkLength(strArg1, strArg2, checkFunc):
    # 해당 함수에서는 문자열의 길이 비교 기준을 모릅니다.
    result = checkFunc(strArg1,strArg2)
    return  result

# 함수 호출시 길이 비교 기준을 정합니다.
min = checkLength(str1, str2, lambda x,y: x if len(x)<len(y) else y)
print(min)

max = checkLength(str1, str2, lambda x,y: x if len(x)>len(y) else y)
print(max)
```


## 사용자 입출력 다루기

#### 사용자 입력받기
- input() 함수를 사용하여 콘솔에서 사용자에게 입력값을 받아올 수 있습니다.
- 해당 코드 위치에서 사용자 입력을 대기합니다. enter 입력시 입력 종료 됩니다.
- 사용자가 입력한 값은 input() 호출된 위치로 반환됩니다.
- input() 함수 호출시 문자열 인수를 전달하면 입력값에 대한 힌트를 제공할 수 있습니다.
- input() 함수를 사용하여 입력받은 값은 모두 str 타입입니다.
- 필요시 원하는 타입으로 형변환 해야합니다.

```
inputValue1 = int(input("숫자1을 입력하세요 : "))
inputValue2 = int(input("숫자2를 입력하세요 : "))
print(inputValue1)
print(type(inputValue1))
print(inputValue2)
print(type(inputValue2))
print(inputValue1+inputValue2)
print(type(inputValue1+inputValue2))
```

#### 출력하기
- print() 함수를 사용하여 콘솔에 값을 출력할 수 있습니다.

```

print('a','b','c') # 'a b c' 로 출력, 콤마는 문자열 사이에 공백을 추가하여 출력합니다.

for i in range(1,11):
    print(i, end = ' ') # end 인수를 사용하면 print 출력 끝 문자를 지정할 수 있습니다. (기본값: \n)
```
