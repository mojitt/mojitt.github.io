## 함수
#### 함수란?
함수는 특정 작업을 수행하는 코드의 집합

주요 목적
- 코드의 재사용성 향상
- 코드의 가독성 향상
- 유지보수 용이
- 기능의 모듈화

```
// 함수의 기본 구조
function 함수명(매개변수1, 매개변수2) {
    // 실행할 코드
    return 반환값;
}
```

#### 일급 함수 (First-Class Function)
- JavaScript의 함수는 일급 객체(First-Class Object)
- 이는 함수를 값처럼 자유롭게 사용할 수 있다는 의미

일급 함수의 특징
- 변수에 할당 : 함수를 변수에 저장 가능, 예) const fn = function() {}
- 객체의 값 : 객체의 속성이나 배열의 요소로 저장, 예) { method: function() {} }
- 매개변수 전달 : 다른 함수의 인수로 전달, 예) setTimeout(fn, 1000)
- 반환값 사용 : 함수가 함수를 반환, 예) return function() {}

## 함수 선언 방식
#### 함수 선언문
- 호이스팅 지원
- 함수 선언문은 코드 실행 전에 먼저 처리되므로, 선언 전에 호출 가능.

```
console.log('=== 1. 함수 선언문 ===');
    
console.log(add(5,5)); // 선언 전 호출

// function 키워드로 선언한 함수는 호이스팅 대상
// 선언 전에도 호출가능 선언 후에도 호출가능
function add(a,b){
    return a+b;
}
console.log(add(10,5)); // 선언 후 호출

// 매개변수 없는 함수 선언
function sayHello(){
    return 'hello';
}
console.log(sayHello());

// 반환값이 없는 함수 (return이 없는 경우)
function func(){
    // return이 없는 경우 블록문의 코드 실행 후 undefined 반환
}
console.log(func());

// 함수 제어
function func2(){
    return ; // 함수를 강제종료하여 return 아래의 코드는 실행되지 않음
    console.log('func2 실행');
}
func2();
```

함수 중복 선언
- 같은 이름의 함수를 여러 번 선언하면 마지막에 선언된 함수로 덮어씌워짐.

```
// 동일명의 함수를 중복 선언 시 
function duplicateFn(){
    console.log('첫번째');
} 
function duplicateFn(){
    console.log('두번째');
}

// 중복된 함수 호출
// 맨 마지막 선언한 함수가 호출
duplicateFn();
```

#### 함수 표현식 (Function Expression)
- 호이스팅 미지원
- 함수 표현식은 변수에 할당되므로, 선언 전에 호출하면 에러가 발생.

```
// function 키워드로 선언한 함수는 호이스팅 지원
// 익명함수로 선언한 함수는 호이스팅 미지원

// console.log(subtract(5,0)); // const 키워드로 선언한 변수는 선언 전에 참조 불가능

const subtract = function(a,b){
    return a-b;
}
/*
중복된 함수명을 피할 수 있다.
const subtract = function(a,b){
    return a-b;
}
*/
console.log(subtract(10,5)); // 익명함수는 선언 후 호출가능
```

#### 화살표 함수 (Arrow Function) - ES6
ES6(ES2015)에서 도입된 간결한 함수 표현 방식.

화살표 함수 장점
- 간결한 문법
- this 바인딩 없음
- 콜백 함수에 적합

화살표 함수 제한
- 생성자로 사용 불가
- arguments 객체 없음
- 메서드로 부적합

```
const arrowFn1 = () => {
    return '화살표함수';
}
// 실행되는 코드가 한줄이고 값을 반환하는 코드라고 하면 한줄로 축약해서 표현 가능
// 블록 생략, return 생략
const arrowFn2 = () => '화살표함수';
console.log(arrowFn1());
console.log(arrowFn2());

// 매개변수가 하나인 경우 괄호를 생략 가능
const arrowFn3 = (name) => {
    return `Hello~! ${name}님`;
}
// 축약된 표현
const arrowFn4 = name => `Hello~! ${name}님`;
console.log(arrowFn3('홍길동'));
console.log(arrowFn4('홍길동'));

// 객체를 반환
const arrowFn5 = name => {
    return { name }
}
// 객체를 반환하는 함수를 축약할 경우 () 묶어서 값을 반환한다.
const arrowFn6 = name => ({ name });
console.log(arrowFn5('홍길동'));
console.log(arrowFn6('홍길동'));

// function vs arrowFunction
const compareObj = {
    name : '홍길동',
    age : 27,
    getName : function(){ // function으로 선언한 함수는 this를 가짐
        return this.name;
    },
    getAge : () => { // 화살표함수로 선언한 함수는 this를 가지지 못함, 가장 가까운 객체인 window 객체를 찾음
        return this;
    }
}
console.log(compareObj.getName());
console.log(compareObj.getAge());
```

## 매개변수(Parameter)와 인수(Argument)
- 매개변수(Parameter): 함수 정의 시 선언하는 변수
- 인수(Argument): 함수 호출 시 전달하는 실제 값

#### 기본 매개변수

```
const introduce = (name, age) => {
    return `이름: ${name}, 나이: ${age}`;
}
// 함수 선언의 매개변수 갯수와 인수의 갯수가 동일할 때 호출
console.log(introduce('홍길동', 27)); // 이름: 홍길동, 나이: 27
// 함수 선언의 매개변수 갯수와 인수의 갯수가 동일하지 않을 때 호출
// 함수 선언의 매개변수와 상관없이 함수 호출 가능
console.log(introduce('홍길동')); // 이름: 홍길동, 나이: undefined
console.log(introduce()); // 이름: undefined, 나이: undefined
```

#### arguments 객체 (일반 함수)
- 일반 함수에서 사용 가능한 유사 배열 객체, 모든 인수를 담고 있음

화살표 함수에는 arguments가 없음!

#### Rest 파라미터 (...rest) - ES6
- Rest 파라미터는 인수들을 실제 배열로 받음.
- 화살표 함수에서도 사용 가능.

```
const restArrowFn = (...rest) => {
    console.log(rest);
}
restArrowFn(1);
restArrowFn(1,2);
restArrowFn(1,2,3);
restArrowFn();

const restArrowFn2 = (name,...rest) => {
    console.log(`이름: ${name}, 취미: ${rest}`);
}
restArrowFn2('홍길동', '영화감상', '음악감상');
```

#### 스프레드 문법 (Spread Syntax)
- 배열이나 객체를 개별 요소로 전개합니다.

```
console.log('==== 스프레드 문법 =====');

// 배열 복사
const arr1 = [100, 200, 300, 400];
const arr2 = [];
for(const element of arr1){
    arr2.push(element);
}
console.log(arr1, arr2);
// 스프레드 문법 사용
const arr3 = [...arr1];
console.log(arr3);
const arr4 = [...arr1, ...arr2];
console.log(arr4);

// 객체 복사
const obj1 = { name: '홍길동', age: 20};
const obj2 = {};
for(const key in obj1){
    obj2[key] = obj1[key];
}
console.log(obj1, obj2);
// 스프레드 문법 사용
const obj3 = { ...obj1 };
console.log(obj3);

// 객체 병합
// 중복된 속성명이 존재할 경우 마지막에 병합하는 객체의 속성으로 덮어씀
const obj4 = { color: 'red', size: 'M'};
const obj5 = { size: 'L', style: 'casual'};
const mergeObj = { ...obj4, ...obj5};
console.log(mergeObj);

// 함수호출문, 매개변수 선언 시 활용
const arrTotal = (...rest) => {
    let total = 0;
    for(const element of rest){
        total += element;
    }
    return total;
}
const targetArr1 = [100, 200, 300, 400];
const total1 = arrTotal(...targetArr1); // arrTotal(100, 200, 300, 400)
console.log(total1);
```


## 구조 분해 할당 (Destructuring)
배열이나 객체의 값을 개별 변수로 쉽게 추출하는 문법.

#### 배열 구조 분해

```
// 나머지 요소의 갯수를 상관없이 배열로 할당할 때
const [nm, ...hobby] = ['홍길동', '영화감상', '음악감상', '독서'];
console.log(nm, hobby);

// 변수 스왑
let x=10, y=20;
[x, y] = [y, x];
console.log(x, y);
```

## 콜백 함수 (Callback Function)

- 비동기 처리 (setTimeout, fetch)
- 배열 메서드 (map, filter, reduce)

```
    console.log('메인함수 호출');

    callback();

main(function(){
    console.log('메인함수 호출 후 실행');
});

// 콜백함수 매개변수
function fetchApi(callback){
    /*
    // 시간이 걸리는 로직
    const processResult = [100,200,300];
    callback(processResult);
    */
    let processResult;
    // 처리시간 : 1초
    setTimeout(function(){
        processResult = [100,200,300];
        callback(processResult);
    }, 1000);
    return processResult;
}
/*
const response = fetchApi();
console.log(response);
*/

fetchApi((response) => {
    console.log(response);
});


// 실습: 숫자 배열요소 2를 곱한 결과 값을 새로운 배열에 담아 배열을 반환하는 함수
```

## 스코프 (Scope)
- 스코프는 변수의 유효 범위를 의미.
- JavaScript는 렉시컬 스코프(Lexical Scope) 방식.

#### 렉시컬 스코프 vs 동적 스코프
렉시컬 스코프 : 함수 정의 시점, 함수가 선언된 위치, JavaScript 사용  
동적 스코프 : 함수 호출 시점, 함수가 호출된 위치, JavaScript 미사용

```
console.log('==== 렉시컬 스코프 vs 동적 스코프 ====');
var variable = 10;

function lexicalFn(){
    var variable = 20;
    testFn1();
}

function testFn1(){
    console.log(variable);
}
lexicalFn();

// 중첩함수
var variable = 10;

function outerFn(){
    var variable = 20;
    innerFn();
    function innerFn(){
        console.log(variable);
    }
}
outerFn();
```

#### 전역 스코프 vs 지역 스코프

```
// 전역 스코프
// var : 함수 스코프
// const, let : 블록 스코프
var globalVar = '전역 변수';

function scopeTest(){
    var globalVar = '지역 변수';
    console.log(globalVar); // 지역변수
}
scopeTest();
/*
var는 함수스코프(유효범위)를 준수하기 때문에
함수를 벗어난 지역에서는 참조할 수 없다.
*/
console.log(globalVar); // 전역변수

if(true){
    let blocklet = 'let으로 선언한 변수';
    const blockconst = 'const로 선언한 변수';
    var globalVar = '블록안에서 var로 선언한 변수';

    console.log(blocklet); // 블록의 유효범위에 있기 참조가능
    console.log(blockconst); // 블록의 유효범위에 있기에 참조가능
}
//console.log(blocklet); // 유효범위를 벗어나서 참조 불가능
//console.log(blockconst); // 유효범위를 벗어나서 참조 불가능
console.log(globalVar);
```

#### 클로저 (Closure)
- 클로저는 함수와 그 함수가 선언된 렉시컬 환경의 조합.
- 내부 함수가 외부 함수의 변수에 접근할 수 있는 것.

```
/*
const numCnt = (num) => {
    return () => {
        ++num;
        return num;
    }
}
```

## 재귀 함수 (Recursive Function)
재귀 함수는 자기 자신을 호출하는 함수.

재귀 함수 장점
- 코드가 간결하고 이해하기 쉬움

재귀 함수 단점
- 성능이 반복문보다 느림
- 스택 메모리 사용 (오버플로우 위험)
- 디버깅이 어려울 수 있음

필수 조건
- 종료 조건(Base Case)이 반드시 있어야 함.
- 종료 조건이 없으면 무한 루프에 빠져 스택 오버플로우 발생!

```
function factorialFn(n){
    if (n===1) return 1;
    return n*factorialFn(n-1);
}
console.log(factorialFn(5));
//factorialFn(5) => 5! = 5*4*3*2*1


// 실습. 배열 요소의 합계
function sumArr(arr, i=0){
    if (i === arr.length) return 0;
    return arr[i] + sumArr(arr, ++i);
}
console.log(sumArr([100,200,300,400]));
```
