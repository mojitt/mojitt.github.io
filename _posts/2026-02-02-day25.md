#### equals() 메소드
- 서로 다른 두 객체의 번지수가 같은지 비교하는 메소드
- 리턴 값은 boolean 타입으로 번지수가 같으면 true 다르면 false를 반환 합니다. 

```
Sample1 sample1 = new Sample1();
Sample1 sample2 = sample1;
Sample1 sample3 = new Sample1();
		
System.out.println(sample1.equals(sample2)); //true
System.out.println(sample1.equals(sample3)); //false
```

- 주로 엔티티 클래스 내에서 필드의 논리적 동등비교를 위하여 재정의하여 사용합니다.
- 예) 주민번호가 같으면 물리적 객체 주소가 서로 다른 객체라도 동등객체로 취급하기 위하여

```
//Person.class

private String jumin;

public Person(String jumin) {
	this.jumin = jumin;
}
	
public String getJumin() {
	return jumin;
}
public void setJumin(String jumin) {
	this.jumin = jumin;
}

@Override
public boolean equals(Object obj) {
	/*
	 <equals 메소드의 재정의 목적>
	 현재 객체와 매개변수 obj에 담긴 객체의 jumin 필드값이 같으면 true 반환!
	 */
	boolean result = false;
	if(obj instanceof Person) {
		Person p = (Person) obj;
		result = this.getJumin().equals(p.getJumin());
	}
	return result;
}
```

```
//Main.class

Person person1 = new Person("123456-1234567");
Person person2 = new Person("123456-1234567");
		
//Person.equals() 메소드를 재정의 안했을때
//System.out.println(person1.equals(person2)); false

//Person.equals() 메소드를 재정의 했을때
System.out.println(person1.equals(person2)); //true
```

#### hashCode 메소드
- 객체의 물리적인 주소를 int로 변환하여 반환하는 메소드
- 자바에서는 hashCode의 결과를 먼저 비교하여 동등객체인지 판별하므로 논리적 동등객체는 hashCode 값 역시 같게 나와야 합니다.
- hashCode 리턴값만으로는 필드마다의 동등비교가 어렵기 때문에 hashCode 메소드를 진행하고 equals 메소드로 검증하여 동등 객체 판단

```
String str1 = new String("A");
String str2 = new String("A");
	
//String 타입은 값의 동등비교를 위하여 hashCode 메소드가 이미 값이 같으면 같은 반환값을 갖도록 재정의 되어있습니다.
System.out.println("str1의 hashCode -> "+str1.hashCode()); //65
System.out.println("str2의 hashCode -> "+str2.hashCode()); //65
System.out.println(str1.hashCode()==str2.hashCode()); //true
```

- 서로 다른 두 객체의 논리적 동등비교를 위하여 재정의하여 사용합니다.

```
//Person.class

@Override
public int hashCode() {
	/*
	 <hashCode 메소드의 재정의 목적>
	 Person 타입 객체들은 jumin 필드 값이 같다면 같은 hashCode 값을 반환하도록 재정의
	여러 값을 비교하고 싶다면 필드값들을 합하여 리턴
	예) return a.hashCode()+b.hashCode()+... ;
	 */
	return jumin.hashCode();
}
```

```
//Main.class

Person person1 = new Person("123456-1234567");
Person person2 = new Person("123456-1234567");

System.out.println("person1.jumin -> "+person1.getJumin()); //123456-1234567
System.out.println("person2.jumin -> "+person2.getJumin()); //123456-1234567

//재정의 안했을 경우
//System.out.println("person1 hashCode -> "+person1.hashCode()); 625576447
//System.out.println("person2 hashCode -> "+person2.hashCode()); 1560911714

//재정의 했을 경우
System.out.println("person1 hashCode -> "+person1.hashCode()); //744922314
System.out.println("person2 hashCode -> "+person2.hashCode()); //744922314
```

#### toString 메소드
- 객체의 주소 정보를 문자열로 반환합니다.
- 주로 엔티티 클래스로 생성된 객체의 전체 필드값을 문자열로 반환하는 용도로 재정의하여 사용합니다.

```
Sample1 sample1 = new Sample1();
Sample1 sample2 = new Sample1();
System.out.println("sample1 toString -> "+sample1.toString()); //java17.Sample1@68de145
System.out.println("sample2 toString -> "+sample2.toString()); //java17.Sample1@27fa135a
```

- String 타입은 toString() 문자열 데이터를 출력하도록 재정의 되어있습니다.

```
//Student.class

@Override
public String toString() {
	return "Student [studentNum=" + studentNum + ", name=" + name + ", age=" + age + ", year=" + year
		+ ", classNum=" + classNum + ", address=" + address + "]";
}
```

```
//Main.class

Student student1 = new Student();

System.out.println("student1 toString -> "+student1.toString());
System.out.println("student1 -> "+student1); 
//객체의 참조 변수를 호출하는 것과 toString()을 호출하는 것은 같은 결과입니다.
```


### System 클래스
- 운영체제의 일부 기능을 사용할 수 있는 클래스
- 대부분 정적 필드, 정적 메소드로 제공
- 출력 : out 필드, 입력 : in 필드
- 시간 정보 필요시 
	- currentTimeMillis() -> 현재 시간을 밀리초로 반환
	- nanoTime() -> 현재 시간을 나노 초 단위로 반환

```
System.out.println("currentTimeMillis() -> "+System.currentTimeMillis()); //long 타입으로 반환, 1770010334489 출력
System.out.println("nanotime() -> "+System.nanoTime()); //long 타입으로 반환, 19962305258700 출력
```

#### StringBuilder 클래스
- 문자열 저장소에 문자열들을 저장한 후 한번에 출력하는 객체 타입
- 문자열이 자주 변하는 경우 사용하면 메모리 관리를 효율적으로 할 수 있습니다.

```
StringBuilder sb = new StringBuilder(); //문자열 저장소 생성
		
sb.append("A");		
System.out.println(sb.toString()); //A
		
sb.append("B");
System.out.println(sb.toString()); //AB
		
sb.append("C");
System.out.println(sb.toString()); //ABC
		
sb.append(" java"); //문자열 맨 뒤에 추가
System.out.println(sb.toString()); //ABC java
		
sb.insert(0, "hello ");
System.out.println(sb.toString()); //hello ABC java

sb.insert(1, "i "); //1번 인덱스에 "i " 삽입
System.out.println(sb.toString()); //hi ello ABC java
		
sb.delete(3, 8); //3번 인덱스부터 7번 인덱스까지 삭제
System.out.println(sb.toString()); //hi ABC java
		
sb.replace(0, 2, "hello"); //0번 인덱스부터 1번 인덱스까지 "hello"로 치환
System.out.println(sb.toString()); //hello ABC java
		
System.out.println(sb); //toString() 호출과 동일, hello ABC java 출력
```

StringTokenizer st = new StringTokenizer(str4, ",");

### 박스클래스 (포장클래스)
- 기본타입 데이터를 객체화 시켜주는 클래스 입니다.
- 박싱 : 기본타입을 객체화 / 언박싱 : 객체를 기본타입으로 변환

포장 클래스들은 해당 기본타입을 사용할 수 있는 여러 정적 멤버들을 제공하고 있습니다.
- 예) Integer.parseInt(문자열), MAX_VALUE, MIN_VALUE

```
byte b = 10;
Byte bBox = b; //박싱
bBox = new Byte((byte)10); //정식 박싱
		
short s = 10;
Short sBox = s; //박싱
sBox = new Short((short)10);
		
char c = 10;
Character cBox = c; //박싱
cBox = new Character((char)10);
		
int i = 10;
Integer iBox = i; //박싱
iBox = new Integer(10);
	
long l = 10;
Long lBox = l; //박싱
lBox = new Long((long)10);
		
float f = 1.0f;
Float fBox = f; //박싱
fBox = new Float((float)1.0f);
		
double d = 1.0;
Double dBox = d; //박싱
dBox = new Double((double)1.0);
	
boolean bl = true;
Boolean blBox = bl; //박싱
blBox = new Boolean(true);

i = iBox.intValue(); //언박싱
System.out.println(i);
		
b = bBox.byteValue(); //언박싱
System.out.println(b);
	
s= sBox.shortValue(); //언박싱
System.out.println(s);
		
c = cBox; //자동 언박싱
System.out.println(c);
		
l = lBox; //자동 언박싱
f = fBox; //자동 언박싱
d = dBox; //자동 언박싱
bl = blBox; //자동 언박싱
```

### Math 클래스
- 수학과 관련된 여러 기능을 제공하는 클래스

#### abs(숫자)
숫자의 절대값 반환

```
int number = -100;
System.out.println(Math.abs(number)); //100
```

#### ceil(실수)
소수점 올림 / 반환 타입 -> double

```
double dNumber = 10.6;
int result = (int)Math.ceil(dNumber);
System.out.println(result); //11
```

#### floor(실수)
소수점 버린 정수 결과를 반환 합니다. / 반환 타입 -> double

```
double dNumber = 10.6;
result = (int)Math.floor(dNumber);
System.out.println(result); //10
```

#### round(실수)
소수점을 반올림한 정수 결과를 반환 합니다 / 반환 타입 -> long

```
double dNumber = 10.6;
long lResult = Math.round(dNumber);
System.out.println(lResult); //11
```
