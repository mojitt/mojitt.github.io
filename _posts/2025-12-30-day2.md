### 정의형 목록태그
```
<dl>
    <dt>주제</dt>
    <dd>설명</dd>
</dl>
```
- 정의하고자 하는 주제가 있고 주제에 대한 설명이 나열되는 항목 구조
- dl 태그 : 정의형 목록 영역 지정
- dt 태그 : 정의형 항목 주제를 작성
- dd 태그 : 주제에 대한 설명을 작성
- 각 주제와 그에 대한 설명 영역은 div로 그룹지을 수 있습니다.

### 태이블 태그
```
<table>
    <caption>표 제목</caption>
    <thead>
        <tr>    
            <th>이름</th>
        </tr>
    </thead>
    <tbody>
        <tr>
            <td>내용</td>
        </tr>
    </tbody>
    <tfoot>
        <tr>
            <td>내용</td>
        </tr>
    </tfoot>
</table>
```
- 본문에 표를 그리기 위한 태그
- caption을 사용하여 표의 제목을 짓습니다.
- thead, tbody, tfoot 을 사용하여 행들의 그룹을 짓습니다.
- 출력되는 순서는 무조건 thead, tbody, tfoot 순으로 출력됩니다.
- tr을 사용하여 행을 생성합니다.
- th, td를 사용하여 셀(칸)을 그립니다.
- th -> 열 제목 (굵게, 가운데 정렬) / td -> 열 데이터 (기본 굵기, 왼쪽 정렬)
- 시멘틱 태그인 thead, tbody, tfoot을 생략하면 브라우저가 강제로 tbody를 끼워 적용합니다.
- css, javascript 작업에 문제가 생기지 않기 위하여 생략하지 않습니다.

### 표 병합하기
가로 병합하기
```
<table>
    <tbody>
        <tr>
            <td colspan="2">1</td>
            <td>3</td>
        </tr>
        <tr>
            <td >4</td>
            <td colspan="2">5</td>
        </tr>
        <tr>
            <td colspan="3">7</td>
        </tr>
    </tbody>
</table>
```
- **가장 왼쪽 셀**의 시작태그에 colspan 속성을 사용하여 가로 병합을 합니다.
- 필요 없는 셀들은 지웁니다.

세로 병합하기
```
<table>
    <tbody>
        <tr>
            <td rowspan="2">1</td>
            <td rowspan="3">2</td>
            <td>3</td>
            <td>4</td>
        </tr>
        <tr>
            <td>7</td>
            <td>8</td>
        </tr>
        <tr>
            <td>9</td>
            <td>11</td>
            <td>12</td>
        </tr>
    </tbody>
</table>
```
- **가장 위에 있는 셀**의 시작 태그에 rowspan 속성을 사용합니다.
- 아래 행들에서 필요 없는 셀은 제거합니다.

가로 세로 병합하기
- 병합하고자 하는 셀들 중 **가장 왼쪽 위에 있는 셀**의 시작태그에 colspan과 rowspan을 함께 사용합니다.
- 필요 없는 셀들은 제거합니다.

### 레이아웃 시멘틱 태그들
```
<body>
    <header>
        <nav></nav>    
    </header>
    <section></section>
    <article></article>
    <aside></aside>
    <footer></footer>
</body>
```
시멘틱 태그를 사용하여 레이아웃 구성하기
- header, nav, section, main, article, aside, footer 태그들을 사용하여 웹페이지의 구조를 지정합니다.
- 필요에 따라 필요한 시멘틱 태그를 사용하여 구조를 잡습니다.
- 전부 블럭성질이며 css를 사용하여 화면 배치를 하게 된다.

### 기타 시멘틱 태그들
```
<details>
    <summary>클릭해보세요.</summary>
    <div>
        클릭하면 펼쳐집니다!!!
    </div>
</details>
```
- details 태그를 사용하여 항목 영역을 지정합니다.
- summary 태그를 사용하여 주제를 지정합니다.
- summary 태그 아래 내용이 브라우저에서 요소 클릭시 펼쳐지거나 접히게합니다.
- details 태그 안에 summary 태그는 하나만 존재합니다.

### 링크 태그
```
<a href="https://mojitt.github.io">현재 창에서 열기</a>
<a href="https://mojitt.github.io" target="_blank">새창에서 열기</a>
```
- 웹 페이지에서 다른 경로로 이동하는 링크를 만드는 태그입니다.
- a 태그 사용
- 인라인 성질이지만 예외적으로 블럭 성질 태그를 중첩할 수 있습니다.
- target 속성을 사용하여 웹페이지 이동 창 제어할 수 있습니다.
- '_blank' 값은 새창에서 웹페이지 이동합니다.
- '_self' 값은 현재 창에서 웹페이지 이동합니다. (기본값)

### 앵커 다루기
```
<a href="#number1">1번으로 이동</a>                         <!--#:현재페이지-->
<a href="#number2">2번으로 이동</a>
<div id="number1" style="height: 300px; border: 1px solid black">
    <h3>1번</h3>
</div>
<div id="number2" style="height: 300px; border: 1px solid black">
    <h3>2번</h3>
</div>
```
- 이동되기를 원하는 경로에 id값을 지정하고, href 속성값으로 이동하고 싶은 id값을 줍니다.

### 미디어 태그
오디오 파일 삽입하기
```
<audio src="sample.mp3" controls loop autoplay></audio>
```
- audio 태그를 사용하여 오디오 파일을 원하는 위치에 삽입할 수 있습니다.
- src 속성으로 삽입하고자 하는 음성 파일 경로를 지정합니다.
- controls 속성으로 해당 위치에 재생제어기를 출력합니다.
- loop 속성으로 자동재생을 지정합니다.
- autoplay 속성으로 자동재생을 지정합니다.(현재는 지원x)

비디오 파일 삽입하기
```
<video src="sample.mp4" controls loop autoplay muted poster="썸네일.png"></video>
```
- video 태그를 사용하여 비디오 파일을 원하는 위치에 삽입할 수 있습니다.
- src 속성으로 비디오 파일 경로를 지정합니다.
- controls 속성으로 비디오 재생제어기를 생성합니다.
- width/height 속성으로 비디오의 너비 높이를 제어합니다.
- loop 속성으로 반복재생을 지정합니다.
- autoplay 속성으로 자동재생을 지원합니다.(음소거 설정이여야 가능)
- muted 속성으로 음소거로 영상 시작을 합니다.
- poster 속성으로 영상 실행 전 썸네일을 지정합니다.

### 웹폼을 다루는 태그
form 태그
```
<form name="frm" action="처리페이지.html" method="get"></form>
```
- 영역 안에서 입력 폼을 통해 입력한 데이터들에 대한 처리 정보를 담는 태그입니다.
- 처리를 위한 데이터 입력 태그들은 반드시 form 태그 안에 존재해야 합니다.
- name 속성을 사용하여 태그의 이름을 짓습니다.
- action 속성을 사용하여 form 태그 안 입력 태그에 작성된 데이터를 처리하는 곳 경로를 작성합니다.
- method 속성을 사용하여 입력한 데이터 전송 방식을 지정합니다.
*get : 입력 데이터를 url 뒤에 붙여서 전송 (검색) / post : 입력 데이터를 숨겨서 안전하게 전송 (로그인, 등록, 수정, 삭제)

### 입력 양식 태그 종류(1)
한 줄 입력 양식
```
<input type="text" name="id" size="30" maxlength="5">
```
- 한 줄 입력할 수 있는 텍스트 입력 양식 태그 입니다.
- size 속성을 사용하여 출력 길이를 제어합니다.
- maxlength 속성을 사용하여 입력 값의 길이를 제어합니다.

비밀번호 입력 양식
```
<input type="password" name="pw" size="10" maxlength="10">
```
- 비밀번호를 입력할 수 있는 텍스트 입력 양식 태그 입니다.
- 작성된 값을 불릿(******) 처리되어 출력됩니다.
- size 속성을 사용하여 출력 길이를 제어합니다.
- maxlength 속성을 사용하여 입력 값의 길이를 제어합니다.

이메일 입력 양식
```
<input type="email" name="email" size="10">
```
- 이메일을 입력할 수 있는 텍스트 입력 양식 태그입니다.
- 전송버튼 클릭시 입력된 값이 이메일 형식인지 확인합니다.
- size 속성을 사용하여 출력 길이를 제어합니다.
- maxlength 속성을 사용하여 입력 값의 길이를 제어합니다.

검색 입력 양식
```
<input type="search" name="search">
```
- 검색어를 입력할 수 있는 텍스트 입력 양식 태그입니다.
- 오른쪽 끝에 x 버튼 제공합니다.
- 모바일이나 태블릿 사용시 검색 버튼 있는 키패드 활성화 합니다.
- size 속성을 사용하여 출력 길이를 제어합니다.
- maxlength 속성을 사용하여 입력 값의 길이를 제어합니다.

숫자 입력 양식
```
<input type="number" name="count" min="0" max="20" step="0.5">
```
- 숫자만 입력할 수 있는 입력 양식 태그 입니다.
- 오른쪽 끝에 스핀버튼을 제공합니다.
- 앞자리 0은 허수로 처리됩니다.
- min 속성으로 입력 최솟값을 지정하며 전송버튼 클릭시 확인합니다.
- max 속성으로 입력 최댓값을 지정하며 전송버튼 클릭시 확인합니다.
- step 속성을 사용하여 스핀버튼의 값 범위를 지정할 수 있습니다.

년월일 입력 양식
```
<input type="date" name="date" min="2025-12-20" max="2026-01-01">
```
- 달력을 제공하여 년, 월, 일을 입력할 수 있는 입력양식입니다.
- min 속성 사용하여 최소날짜 지정합니다.
- max 속성 사용하여 최대날짜 지정합니다.

시간 입력 양식
```
<input type="time" name="time">
```
- 시계를 제공하여 시, 분, 오전, 오후를 입력할 수 있는 입력 양식입니다.

년월일 시간 입력 양식
```
<input type="datetime-local" name="datetime">
```
- 달력을 제공하여 년, 월, 일, 시, 분을 입력할 수 있는 입력 양식입니다.

데이터 리스트 입력 양식
```
<input type="text" list="textlist">
<datalist id="textlist">
    <option>apple</option>
    <option>banana</option>
    <option>cat</option>
    <option>air</option>
    <option>html</option>
    <option>css</option>
</datalist>
```
- 리스트를 미리 작성하여 리스트를 펼쳤을 때 선택할 수 있는 입력 양식입니다.

라디오 버튼 입력 양식
```
<input type="radio" name="rbtn" value="o"> 버튼1
<input type="radio" name="rbtn" value="x"> 버튼2
```
- name 속성이 같으면 같은 그룹의 라디오 버튼들이 됩니다.
- 각 체크박스 항목에 value 속성값을 작성해야 처리하는 곳에서 선택된 체크박스 값을 알 수 있습니다.
