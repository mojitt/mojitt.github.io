#### Faker
테스트용 가짜 데이터를 생성할 때 사용하는 라이브러리

```
from faker import Faker

faker = Faker('ko-KR')

for i in range(1,100):
    print(faker.name())
    print(faker.address())
    print(faker.postcode())
```

#### 클로저
- 함수 안에 내부 함수를 정의하여 반환하는 것을 의미합니다.
- 함수의 지역변수 값을 기억하기 위하여 주로 사용합니다.

```
def func1():
    def func2():
        print('내부함수 실행 중')
        return 10
    return func2

var1 = func1()

result = var1()
print(result)

def multi(num1):
    print(num1)
    def wrapper(num2):
        print(num2)
        return num1*num2
    return wrapper

mul1 = multi(3)

result = mul1(5)
print(result)

def mulOf7():
    num1 = 7
    def gugu():
        for i in range(1,10):
            print(f'{num1} * {i} = {num1*i}')
    return gugu

result = mulOf7()
result()

# 외부함수의 변수 값을 내부함수에서 수정하거나 쓰기를 하기 위해서는 
# nonlocal 키워드를 사용하여 외부함수 변수임을 명시해야합니다.
def func3():
    value = 10
    def func4():
        nonlocal value
        value += 20
        return value
    return func4

result = func3()
print(result()) # +20 = 30
print(result()) # +20 = 50
print(result()) # +20 = 70

# 외부함수의 변수가 반복객체일 경우 객체 내부를 수정하는 것은
# nonlocal 키워드 없이 가능합니다.
def list_c():
    value = []
    def add(arg):
        value.append(arg)
        return value
    return add

list_name = list_c()
print(list_name('홍길동'))
print(list_name('고길동'))
```

#### 데코레이터
- runtime 시에 함수에 로그 남기다던지 추가 처리를 위하여 호출되는 함수를 의미한다.
- 주기능 함수가 호출되기 전에 추가처리하는 함수를 말합니다.
- 데코레이터 정의 시 내부 함수의 매개변수는 *args, **kwargs 로 정의되어 있어야 모든 함수에서 사용할 수 있습니다.
- *어떤 함수에서 해당 데코레이터를 사용할지 알 수 없으므로 모든 매개변수를 대응하기 위하여

```
from dataclasses import dataclass

def deco1(func1):
    def wrapper(*args1, **args2):
        print('주 기능 함수 호출 전 입니다.')
        func1(*args1, **args2)
    return wrapper

check_func = deco1(lambda x : x+10)
check_func(10)

print('-'*40)

@deco1
def func2(arg1, arg2):
    print(f'{arg1}입니다.')
    print(f'{arg2}입니다.')

func2(10,20)

print('='*40)

class User:
    def __init__(self):
        self.name1 = ''
    
    @property # getter 정의 데코레이터    
    def name(self):
        return self.name1
    
    @name.setter # setter 정의 데코레이터
    def name(self, value):
        self.name1 = value
        
hong = User()
hong.name = '홍길동'
print(hong.name)

@dataclass
class Person:
    name : str
    age : int

person1 = Person('홍길동', 30)
print(person1)
print(person1.name)
print(person1.age)
```
