## 리액트란?
- 웹과 네이티브 사용자 인터페이스를 위한 라이브러리
- 사용자 인터페이스 (User Interface, UI)
	- 사용자와 컴퓨터 프로그램의 상호작용을 위해 중간에서 입력과 출력을 제어해주는 역할
	- 예: 버튼이나 텍스트 입력창 등
- 사용자 인터페이스를 만들기 위한 기능 모음집(라이브러리)을 UI 라이브러리라고 부름
- 리액트는 대표적인 자바스크립트 UI 라이브러리

#### 리액트의 장점
- 빠른 업데이트와 렌더링 속도
- 재사용성이 높은 컴포넌트 기반 구조
- 메타(구 페이스북)의 든든한 지원
- 활발한 지식 공유와 커뮤니티
- 리액트 네이티브를 통해 모바일 앱 개발 기능

#### 리액트의 단점
- 방대한 학습량
- 높은 상태 관리 복잡도

#### JSX란?
JavaScript와 XML/HTML을 함께 사용할 수 있는 자바스크립트의 확장 문법

#### JSX의 역할
- JSX로 작성된 코드는 모두 자바스크립트 코드로 변환됨
- 리액트는 JSX 코드를 모두 createElement() 함수를 사용하는 코드로 변환

#### JSX의 장점
- 코드가 간결해짐
- 가독성이 향상됨
- Injection Attack을 방어함으로써 보안성이 올라감

#### JSX 사용법
- 기본적으로 모든 자바스크립트 문법을 지원함
- 자바스크립트에 XML과 HTML을 섞어서 사용
- 중괄호를 사용하여 자바스크립트 코드를 삽입

## Element
#### element란?
- 리액트 앱의 가장 작은 빌딩 블록들
- 화면에 나타나는 내용을 기술하는 자바스크립트 객체
- 리액트 엘리먼트는 DOM 엘리먼트의 가상표현

#### element 특징
- 불변성을 갖고 있음
- 엘리먼트 생성 후에는 자식이나 속성을 바꿀 수 없음

```
const Element01 = () => {
    let time_data = new Date().toLocaleTimeString()

    /*
    setInterval(() => {
        time_data = new Date().toLocaleTimeString()
    },1000);
    */
   
    return (
        <h2>현재 시간: {time_data}</h2>
    )
}

export default Element01
```

## Component
#### 리액트 컴포넌트
- 컴포넌트 기반 구조
- 개념적으로 자바스크립트의 함수와 비슷

#### Props의 개념
- 리액트 컴포넌트의 속성
- 컴포넌트에 전달한 다양한 정보를 담고 있는 자바스크립트 객체

#### Props의 특징
- 읽기 전용 (Read-Only)
- 리액트 컴포넌트의 props는 바꿀 수 없고, 같은 props가 들어오면 항상 같은 엘리먼트를 리턴해야 한다.

#### 컴포넌트 만들기
- 함수 컴포넌트
- 클래스 컴포넌트

```
// Component01.jsx
const Component01 = (props) => {
    
    // 컴포넌트 내부에서 props는 수정할 수 없습니다.
    // 컴포넌트 pure 해야하기 때문에 (입력 정보가 같은 컴포넌트는 출력 컴포넌트도 같아야 한다.)
    //props.title = '변경할거야!'

    return (
        <div>
            <p>{props.title}[{props.hit}]</p>
            {props.children}
        </div>
    )
}

export default Component01

// App.jsx
function App() {
  return (
    <>
      <Component01 title="component1" hit={100}>
        <h4>안녕, 나는 컴포넌트1이야</h4>
      </Component01>
      <Component01 title="component2" hit={200}>
        <h5>안녕, 나는 컴포넌트2이야</h5>
      </Component01>
      <Component01 title="component3" hit={300}>
        <h6>안녕, 나는 컴포넌트3이야</h6>
      </Component01>
    </>
  )
}
```

import React from "react";
