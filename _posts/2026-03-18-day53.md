### 이벤트 객체 (Event Object)
- 이벤트가 발생하면 이벤트 관련 정보를 담은 이벤트 객체가 동적으로 생성됨.
- 생성된 이벤트 객체는 이벤트 핸들러의 첫 번째 인수(매개변수)로 자동 전달됨.
- 어트리뷰트 방식은 이벤트 객체를 event라는 이름으로만 참조 가능 (암묵적 매개변수명 고정).

[ 이벤트 객체 주요 프로퍼티 / 메서드 ]

| 구분 | 이름 | 설명 |
| :--- | :--- | :--- |
| 프로퍼티 | `event.type` | 발생한 이벤트 타입 문자열 (`'click'`, `'input'` 등) |
| | `event.target` | 실제로 이벤트가 발생한 요소 (클릭된 자식 요소) |
| | `event.currentTarget` | 이벤트 핸들러가 등록된 요소. 일반 함수에서는 `this` 와 동일. |
| | `event.key` | 키보드 이벤트에서 입력된 키 값 (예: `'Enter'`, `'a'`) |
| | `event.bubbles` | 이벤트 버블링 여부 (boolean) |
| 메서드 | `event.preventDefault()` | 태그의 기본 동작 방지 (a 태그 페이지 이동, submit 전송 등) |
| | `event.stopPropagation()` | 이벤트 버블링(상위 전파) 중단 |

```
const $eventBtn = document.getElementById('eventBtn');
// 클릭이 일어났을 때 자바스크립트엔진이 생성된 이벤트 객체를
```

### 폼 이벤트 실습 — 회원가입 유효성 검사
- blur: 포커스를 잃을 때 즉시 유효성 검사 실행.
- change: 비밀번호 변경 시 비밀번호 확인 필드를 자동 초기화.
- closest(selector): 가장 가까운 상위 요소를 탐색
- $form.submit(): 폼 요소의 submit 메서드를 직접 호출하여 전송.

```
const validation = (input, span, isValid) => {
    if(isValid){
        input.classList.add('success');
        input.classList.remove('error');
        span.classList.remove('show');
    }else{
        input.classList.remove('success');
        input.classList.add('error');
        span.classList.add('show');
    }
}

const validate = {
    'memberId' : {
        rule : /^[a-z][a-z0-9]{4,19}$/,
        errorMsg : '회원아이디는 5글자 이상 소문자로 작성해야합니다.'
    },
    'memberPw' : {
        rule: /^[A-Za-z0-9`\-=\\\[\];',\./~!@#\$%\^&\*\(\)_\+|\{\}:"<>\?]{8,16}$/,
        errorMsg : '회원비밀번호는 8글자 이상으로 작성해야합니다.'
    },
    'memberName' : {
        rule : /^[가-힣]+$/,
        errorMsg : '회원이름은 한글로 작성해야합니다.'
    },
}

const $confirmBtn = document.getElementById('confirmBtn');
const $formResult = document.getElementById('formResult');

$confirmBtn.addEventListener('click', e => {
    $formResult.style.display = 'none';

    const $inputs = document.querySelectorAll('#myForm input');
    const ckValid = [...$inputs].map(($input, idx, arr) => {
        const id = $input.id;
        const value = $input.value;
        const validObj = validate[$input.id];
        let isValid = false;

        if(validObj){
            isValid = validObj.rule.test(value);
            $input.nextElementSibling.textContent = validObj.errorMsg;
        } else {
            $memberPw = arr.find($input => $input.id === 'memberPw');
            isValid = Boolean(value) && ($memberPw.value === value);
        }
        validation($input, $input.nextElementSibling, isValid);
        return isValid;
    });
    if(ckValid.includes(false)) return;
    document.querySelector('#myForm').submit();
    $formResult.style.display = 'block';
    $formResult.textContent = [...$inputs].filter($input => $input.id !== 'chekckPw')
                                            .map($input => `${$input.id}: ${$input.value}`).join('\n');
});
```

### 이벤트 버블링과 캡처링 (Event Bubbling & Capturing)
- 이벤트 전파(Event Propagation): 이벤트 발생 시 DOM 트리를 따라 이벤트가 전파되는 것.

- ① 캡처링 단계: window → document → … → 이벤트 발생 요소 방향으로 하향 전파.
- ② 타깃 단계: 이벤트가 실제로 발생한 요소에 도달.
- ③ 버블링 단계: 이벤트 발생 요소 → … → document → window 방향으로 상향 전파.

- addEventListener의 세 번째 인수를 true로 설정하면 캡처링 단계에서 핸들러 실행 (기본값: false = 버블링 단계).
- event.stopPropagation(): 이벤트 전파(버블링/캡처링) 즉시 중단.

[ 이벤트 전파 3단계 ]

| 단계 | 전파 방향 | 설명 |
|------|----------|------|
| ① 캡처링 (Capturing) | window → 이벤트 발생 요소 | `addEventListener('click', fn, true)` 로 감지 |
| ② 타깃 (Target) | 이벤트 발생 요소 | `target`과 `currentTarget`이 동일 |
| ③ 버블링 (Bubbling) | 이벤트 발생 요소 → window | 기본값, 대부분의 이벤트에서 발생 (`focus` / `blur` 제외) |

단계 1. 버블링 직접 체험 — 클릭 로그 확인
- 아래의 주황색 자식 박스를 클릭하면 버블링이 일어나는 것을 확인할 수 있음.
- 자식 클릭 → 자식 핸들러 실행 → 부모 핸들러 실행 순으로 로그가 출력됨.

```
const $bubbleParent = document.getElementById('bubbleParent');
const $bubbleChild = document.getElementById('bubbleChild');
const $bubbleLog = document.getElementById('bubbleLog');
const logs = [];

$bubbleChild.addEventListener('click', e => {
    logs.splice(0);
    logs.push(`[자식] currentTarget: ${e.currentTarget.id}, target: ${e.target.id}\n`);
});
$bubbleParent.addEventListener('click', e => {
    if(e.currentTarget === e.target) logs.splice(0);
    logs.push(`[부모] currentTarget: ${e.currentTarget.id}, target: ${e.target.id}\n`);
    $bubbleLog.textContent = logs.join('');
});
```

단계 2. stopPropagation — 버블링 차단
- event.stopPropagation()을 호출하면 해당 시점에서 상위로의 이벤트 전파가 중단됨.
- 아래 버튼 클릭 시 부모까지 전파되지 않는 것을 콘솔에서 확인.

```
const $stopChildBtn = document.getElementById('stopChildBtn');
const $stopParent = document.getElementById('stopParent');

$stopChildBtn.addEventListener('click', e => {
    e.preventDefault(); // 기본 동작 취소
    //e.stopPropagation(); // 전파 중지

    logs.splice(0);
    logs.push(`[자식] currentTarget: ${e.currentTarget.id}, target: ${e.target.id}\n`);
    console.log(logs.join(''));
});
$stopParent.addEventListener('click', e => {
    if(e.currentTarget === e.target){
        logs.splice(0);  
        logs.push(`[부모] currentTarget: ${e.currentTarget.id}, target: ${e.target.id}\n`);
        console.log(logs.join(''));
    }else if(e.eventPhase === 3){
        logs.push(`[부모] currentTarget: ${e.currentTarget.id}, target: ${e.target.id}\n`);
        console.log(logs.join(''));
    }
});
```

### 이벤트 위임 (Event Delegation) — 동적 바인딩
- 이벤트 위임: 개별 자식 요소에 핸들러를 등록하는 대신, 공통 부모 요소에 핸들러를 등록하고
- 버블링을 이용해 자식 이벤트를 처리하는 패턴.

- 동적 바인딩: JS로 동적 생성된 요소에는 이미 등록한 이벤트가 적용되지 않음.
- → 부모에 이벤트를 위임하면 나중에 생성된 자식 요소에도 자동으로 이벤트가 동작.

- event.target.classList.contains('클래스명')으로 실제 클릭된 요소를 식별.
- closest(selector): 이벤트 발생 요소에서 가장 가까운 상위 요소를 탐색.

[ 이벤트 위임 장점 ]

| 구분 | 개별 등록 | 이벤트 위임 |
|------|-----------|--------------|
| 핸들러 수 | 요소 개수만큼 핸들러 등록 | 부모 요소 하나에만 등록 |
| 동적 요소 | 새로 추가된 요소에 적용 안 됨 | 새로 추가된 요소에도 자동 적용 |
| 메모리 | 핸들러 많아질수록 메모리 증가 | 핸들러 1개로 메모리 효율 좋음 |

```
const $dynamicDiv = document.querySelector('#dynamicDiv');
// addEventListener : 이벤트 중복등록

const addRow = (e) => {
    if(!e.target.classList.contains('copy-btn')) return ;
    const target = e.target;
    const currentTarget = e.currentTarget;
    const $contents = target.closest('.contents');
    const copyNode = $contents.cloneNode(true);
    copyNode.querySelector('input').value = '';
    currentTarget.appendChild(copyNode);
}
const delRow = (e) => {

    const currentTarget = e.currentTarget;
    const $contents = target.closest('.contents');
    if(currentTarget.children.length > 1) $contents.remove();
}
$dynamicDiv.addEventListener('click', addRow);
$dynamicDiv.addEventListener('click', delRow);
```
