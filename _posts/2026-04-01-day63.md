## 이벤트란?
사용자가 버튼을 클릭하는 등의 사건을 의미

#### 이벤트처리하기
- DOM의 이벤트
	- 이벤트의 이름을 모두 소문자로 표기
	- 이벤트를 처리할 함수를 문자열로 전달

- 리액트의 이벤트
	- 이벤트의 이름을 카멜 표기법으로 표기
	- 이벤트를 처리할 함수를 함수 그대로 전달

- 이벤트 핸들러
	- 이벤트가 발생했을 때 해당 이벤트를 처리하는 함수
	- 이벤트 리스너라고 부르기도 함

- 클래스 컴포넌트
	- 클래스의 함수로 정의하고 생성자에서 바이딩해서 사용

- 함수 컴포넌트
	- 함수 안에 함수로 정의하거나 arrow function을 사용해서 정의

## Arguments 전달하기
#### Argument란?
- 함수에 전달할 데이터
- 파라미터 또는 매개변수라고 부르기도 함

#### 클래스 컴포넌트
- arrow function을 사용하거나 Function.prototype.bind를 사용해서 전달

#### 함수 컴포넌트
- 이벤트 핸들러 호출시 원하는 순서대로 매개변수를 넣어서 사용

## 렌더링
#### 조건부 렌더링
- 조건에 따라 렌더링의 결과가 달라지도록 하는 것

#### 엘리멘트 변수
- 리액트 엘리멘트를 변수처럼 저장해서 사용하는 방법

#### 인라인 조건
- 조건문을 코드 안에 집어넣는 것
- 인라인 if
	- if문을 필요한 곳에 작접 넣어서 사용하는 방법
	- 논리 연산자 &&를 사용 (AND연산)
	- 앞에 나오는 조건문이 true일 경우에만 뒤에 나오는 엘리먼트를 렌더링
- 인라인 if-Else
	- if-Else문을 필요한 곳에 직접 넣어서 사용하는 방법
	- 삼항 연산자 ?를 사용
	- 앞에 나오는 조건문이 true면 첫 번째 항목을 리턴, false면 두 번째 항목을 리턴
	- 조건에 따라 각기 다른 엘리멘트를 렌더링하고 싶을 때 사용
- 컴포넌트 렌더링 막기
	- 리액트에서는 null을 리턴하면 렌더링되지 않음
	- 특정 컴포넌트를 렌더링하고 싶지 않을 경우 null을 리턴하면 됨

## 리스트와 키
#### 리스트
같은 아이템을 순서대로 모아놓은 것

#### 키
각 객체나 아이템을 구분할 수 있는 고유한 값

#### 여러 개의 컴포넌트 렌더링
- 자바스크립트 배열의 map() 함수를 사용
	- 배열에 들어있는 각 변수에 어떤 처리를 한 뒤 결과(엘리먼트)를 배열로 만들어서 리턴함
	- map() 함수 안에 있는 엘리먼트는 꼭 키가 필요함

####  리스트의 키
- 리스트에서 아이템을 구분하기 위한 고유한 문자열
- 리스트에서 어떤 아이템이 변경, 추가, 또는 제거되었는지 구분하기 위해 사용
- 리액트에서는 키의 값은 같은 리스트에 있는 엘리멘트 사이에서만 고유한 값이면 됨

#### 다양한 키 값의 사용법
- 숫자 값을 사용
	- 배열에 중복된 숫자가 들어있다면 키값도 중복되기 때문에 고유해야 한다는 키값의 조건이 충족되지 않음
- id를 사용
	- id의 의미 자체가 고유한 값이므로 키값으로 사요하기 적합
	- id가 있는 경우에는 보통 id값을 키값으로 사용
- 인덱스 사용
	- 배열에서 아이템의 순서가 바뀔 수 있는 경우에는 키값으로 인덱스를 사용하는 것을 권장하지 ㅇ낳음
	- 리액트에서는 키를 명시적으로 넣어주지 않으면 기본적으로 이 인덱스 값을 키값으로 사용

```
import { useState } from "react"

const ListCompo = () => {

    const [sArr, setSArr] = useState([{name:'홍길동', id:'hong'}
                                     ,{name:'고길동', id:'go'}
                                     ,{name:'김길동', id:'kim'}]);

    const [id, setId] = useState(''); // 사용자가 입력하는 아이디 담을 그릇
    const [name, setName] = useState(''); // 사용자 입력하는 이름 담을 그릇

    const handlerSArr = () => {
        const inputId = id; // add 버튼 클릭시 id 양식 입력값
        const inputName = name; // add 버튼 클릭시 name 양식 입력값
        setSArr([...sArr,{name:inputName, id:inputId}])

    }

    const handlerDelete = (delId) => {
        let filterArr = sArr.filter((item)=>{
            return item.id != delId // 삭제버튼을 누르지 않으면 true
        });
        setSArr(filterArr)
    }

    let arr = [1,2,3,4,5]

    let eleArr = [] // arr 배열에 대한 엘리먼트 목록을 담을 그릇

    for(let i=0; i<arr.length;i++){
        let value = arr[i]
        let ele = <li>{value}</li>
        eleArr.push(ele)
    }

    let textArr = ['HTML', 'CSS', 'JAVASCRIPT', 'REACT', 'JQUERY']

    let eleArr2 = []

    for(let i=0; i<textArr.length; i++){
        let value = textArr[i]
        let ele = <h3>{value}</h3>
        eleArr2.push(ele)
    }

    return (
        <>
            <ul>
                {/*eleArr (for문을 사용한 element 출력)*/}
                {/* map 사용하여 출력하기 */}
                {arr.map((item, idx/*(), arrObj)*/)=>{
                    return <li key={`arr_${idx}`}>{item}</li>
                })}
                {/* key는 하나의 반복에서 각 엘리먼트를 구분하기위한 값이다. (반복이 다르면 중복 상관 x) */}
                {arr.map((item, idx/*(), arrObj)*/)=>{
                    return <li key={`arr_${idx}`}>{item}</li>
                })}
            </ul>
            <div>
                {/*eleArr2*/}
                {textArr.map((item, idx)=>{
                    return <h3 key={`arr_${idx}`}>{item}</h3>
                })}
            </div>
            <div>
                <h2>state 사용하기</h2>
                <div>
                    id : <input type="text" onChange={(e)=>{setId(e.target.value)}}/> /
                    name : <input type="text" onChange={(e)=>{setName(e.target.value)}}/>
                    <button onClick={handlerSArr} style={{margin:"0px 5px"}}>add</button>
                </div>
                <div>
                    {sArr.map(item=>{
                        return (
                            <p key={item.id}>
                                {item.id} / {item.name}
                                <button onClick={()=>{handlerDelete(item.id)}} style={{margin:"0px 5px"}}>삭제</button>
                            </p>
                        )
                    })}
                </div>
            </div>
        </>
    )
}
export default ListCompo
```
